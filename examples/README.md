# /examples

A handful of fictional artifacts that show how the workspace reads end-to-end:

- `example-solution-PROJ-101.md` → an `artifacts/solutions/` doc
- `example-ADR-001-permission-model.md` → a `knowledge/architecture/` ADR

Use these to learn the style, then **delete the entire `examples/` folder** so it doesn't pollute your real content.
