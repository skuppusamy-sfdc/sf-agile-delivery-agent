## Description

As a Provider Network Specialist, I need a Provider 360 FlexCard summary panel on the Provider__c (Person Account) record page that displays the provider's active Provider_Network_Agreement__c details, associated Fee_Schedule__c, NPI, TIN, and credentialing status so that I can quickly assess network standing without navigating multiple related lists.

## Acceptance Criteria

Given a Provider Network Specialist opens a Provider__c record,
When the Provider 360 FlexCard loads,
Then the panel displays Provider__c.NPI__c, Provider__c.TIN__c, Provider__c.Specialty__c, and Provider__c.Credentialing_Status__c in the header section.

Given the provider has an active Provider_Network_Agreement__c,
When the PNA section renders,
Then the FlexCard shows PNA__c.Agreement_Effective_Date__c, PNA__c.Agreement_Expiration_Date__c, PNA__c.Network_Tier__c, and PNA__c.Contract_Status__c where Contract_Status__c = 'Active'.

Given the provider has an associated Fee_Schedule__c,
When the Fee Schedule section renders,
Then the FlexCard displays Fee_Schedule__c.Schedule_Name__c, Fee_Schedule__c.Effective_Date__c, Fee_Schedule__c.LOB__c, and a count of associated CPT code line items.

-Given the provider has multiple historical PNAs, When the FlexCard loads, Then all historical PNAs are displayed in a scrollable list.- (Deprecated: Only the single most recent Active PNA is shown; historical view moved to separate 'PNA History' tab per design review.)

Given the provider's Credentialing_Case__c.Status__c = 'Expired' or 'Suspended',
When the FlexCard header renders,
Then a red warning badge 'Credentialing Alert' is displayed prominently above the NPI/TIN fields.

## Solution

Build OmniStudio FlexCard 'Provider_360_Summary_FC' embedded on Provider__c Lightning Record Page via App Builder. DataRaptor Extract 'DR_Get_Provider_360' queries Provider__c, related Provider_Network_Agreement__c (filtered to Contract_Status__c = 'Active', ordered by Agreement_Effective_Date__c DESC LIMIT 1), Fee_Schedule__c, and Credentialing_Case__c. Conditional state on FlexCard card set to display 'Credentialing Alert' badge when Credentialing_Status__c IN ('Expired', 'Suspended'). CPT line item count sourced via aggregate SOQL in a separate DataRaptor.
