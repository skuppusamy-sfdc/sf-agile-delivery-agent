## Description

As a Claims Operations Manager, I need the system to automatically generate an Explanation_of_Benefits__c (EOB) record and deliver a formatted EOB document to the policyholder via DocuSign when a Claim__c transitions to 'Adjudicated' status, so that policyholders receive timely, compliant benefit summaries and we reduce manual document handling overhead.

## Acceptance Criteria

Given a Claim__c record's Status__c is updated to 'Adjudicated',
When the record save completes,
Then a new Explanation_of_Benefits__c record is created with EOB__c.Claim__c = Claim__c.Id, EOB__c.Policy__c = Claim__c.Policy__c, EOB__c.Total_Billed_Amount__c = SUM(Claim_Line__c.Billed_Amount__c), EOB__c.Total_Allowed_Amount__c = SUM(Claim_Line__c.Allowed_Amount__c), EOB__c.Total_Member_Responsibility__c = SUM(Claim_Line__c.Member_Responsibility__c), and EOB__c.Status__c = 'Generated'.

Given the EOB__c record is created,
When EOB__c.Status__c = 'Generated',
Then a Conga Composer template ('EOB_Standard_v2') is invoked via Apex to produce a PDF attached to the EOB__c record as a Salesforce File (ContentDocument).

Given the EOB PDF is generated and attached,
When the file is ready,
Then a DocuSign envelope is sent to the policyholder email (Policy__c.Policyholder_Email__c) with the EOB PDF as a view-only attachment (no signature required) and DocuSign envelope status is written back to EOB__c.DocuSign_Status__c.

Given a Claim__c has COB (Claim__c.COB_Applicable__c = TRUE),
When the EOB is generated,
Then the EOB__c record includes EOB__c.COB_Other_Payer_Amount__c populated from Claim__c.COB_Other_Payer_Paid__c and the PDF template renders a COB section.

Given the DocuSign delivery fails (bounce or invalid email),
When the failure event is received via DocuSign webhook,
Then EOB__c.DocuSign_Status__c = 'Delivery Failed' and a high-priority Case is auto-created linked to the Policy__c record with Subject = 'EOB Delivery Failed — Manual Outreach Required'.

Given the EOB is for a TPA-administered policy (Policy__c.TPA_Administered__c = TRUE),
When the EOB is generated,
Then a copy of the EOB PDF is also shared to the TPA's linked Community User via Salesforce Files Sharing, and EOB__c.TPA_Notified__c = TRUE.

## Solution

Record-Triggered Flow on Claim__c (After Save, Status__c changes to 'Adjudicated') invokes an Apex Action ('EOBGenerationService') that queries all Claim_Line__c child records, calculates aggregate amounts, and creates the EOB__c record. Conga Composer is called via REST API from Apex using a named credential. DocuSign eSignature Apex Toolkit sends the envelope. DocuSign webhook payload handled by a public-facing Apex REST endpoint that updates EOB__c.DocuSign_Status__c via a Platform Event (DocuSign_Status_Update__e). TPA file sharing uses ContentDocumentLink insert with ShareType = 'V'. All Apex within a try-catch with custom exception logging to Integration_Log__c.
