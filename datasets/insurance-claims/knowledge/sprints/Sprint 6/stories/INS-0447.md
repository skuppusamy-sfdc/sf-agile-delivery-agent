## Description

As a Provider Relations Specialist, I need a Provider 360 FlexCard on the Provider__c (Person Account) record page that consolidates the provider's active Provider_Network_Agreement__c (PNA) status, linked Fee_Schedule__c details, and any open Credentialing_Case__c records so that I have a single unified view without navigating across multiple related lists.

## Acceptance Criteria

Given a Provider Relations Specialist opens a Provider__c record,
When the Provider 360 FlexCard loads,
Then it displays the most recent Provider_Network_Agreement__c where PNA__c.Provider__c = Provider__c.Id and PNA__c.Status__c IN ('Active', 'Pending Execution'), showing fields: PNA__c.Effective_Date__c, PNA__c.Expiration_Date__c, PNA__c.Network_Tier__c.

Given the PNA is within 90 days of expiration (PNA__c.Expiration_Date__c <= TODAY+90),
When the FlexCard renders,
Then the PNA section displays a yellow warning banner: 'Contract Expiring Soon — Action Required'.

Given the Provider__c has a linked Fee_Schedule__c via PNA__c.Fee_Schedule__c,
When the FlexCard loads,
Then the Fee Schedule section displays Fee_Schedule__c.Schedule_Name__c, Fee_Schedule__c.Effective_Date__c, and Fee_Schedule__c.LOB__c.

Given one or more Credentialing_Case__c records exist where Credentialing_Case__c.Provider__c = Provider__c.Id and Credentialing_Case__c.Status__c NOT IN ('Closed', 'Withdrawn'),
When the FlexCard loads,
Then each open Credentialing_Case__c is listed with fields: Credentialing_Case__c.Case_Number__c, Credentialing_Case__c.Type__c, Credentialing_Case__c.Status__c, Credentialing_Case__c.Assigned_To__c.

Given the provider's NPI (Provider__c.NPI__c) is flagged as excluded in the OIG/SAM integration,
When the FlexCard loads,
Then a red 'OIG Exclusion Alert' banner is prominently displayed above all other sections.

## Solution

Build an OmniStudio FlexCard ('Provider360Card') with three child card states: PNA Summary, Fee Schedule Summary, and Credentialing Cases list. Three DataRaptor Extract sources handle each data domain. PNA expiration warning uses a FlexCard Conditional State with a formula comparing PNA__c.Expiration_Date__c to TODAY+90. OIG exclusion flag sourced from Provider__c.OIG_Excluded__c (Boolean), set by a scheduled Apex batch job integrating with the CMS OIG API. FlexCard deployed to the Provider__c Lightning Record Page via App Builder.
