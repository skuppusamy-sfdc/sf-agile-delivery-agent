## Description

As a Provider Network Specialist, I need a Provider 360 FlexCard that surfaces active Provider_Network_Agreement__c records and associated Fee_Schedule__c details for a given provider so that I can quickly assess a provider's contracted status, effective dates, and applicable fee schedules without navigating multiple related lists.

## Acceptance Criteria

Given a Provider Network Specialist opens a Provider__c (Person Account) record,
When the Provider 360 FlexCard loads,
Then all Provider_Network_Agreement__c records where PNA__c.Provider__c = current Provider__c and PNA__c.Status__c IN ('Active', 'Pending Renewal') are displayed, showing PNA__c.Agreement_Number__c, PNA__c.Effective_Date__c, PNA__c.Termination_Date__c, PNA__c.Network__c, and PNA__c.LOB__c.

Given a PNA record is displayed in the FlexCard,
When the user expands a PNA row,
Then all linked Fee_Schedule__c records (Fee_Schedule__c.PNA__c = PNA__c.Id) are shown with Fee_Schedule__c.CPT_Code__c, Fee_Schedule__c.Allowed_Amount__c, Fee_Schedule__c.Effective_Date__c.

Given a PNA__c.Termination_Date__c is within 30 days of today,
When the FlexCard renders that PNA row,
Then a warning badge 'Expiring Soon' is displayed in amber using the SLDS badge component.

Given the provider has no active PNA records,
When the FlexCard loads,
Then an empty state message is displayed: 'No active network agreements found. Initiate contracting via Provider Network Contracting flow.'

-Given the FlexCard shows TIN and NPI fields inline on the PNA row- (deprecated — TIN/NPI display moved to separate Provider Data card per security review INS-SEC-044)

## Solution

Build OmniStudio FlexCard (ProviderNetworkAgreement_FC) with primary DataRaptor Extract (DR_GetProviderPNAs) querying Provider_Network_Agreement__c filtered by Provider__c and Status__c. Child DataRaptor (DR_GetFeeScheduleByPNA) fetches Fee_Schedule__c records, triggered on row expand action. Expiry badge logic implemented via FlexCard Conditional State on a formula field comparing PNA__c.Termination_Date__c to TODAY(). FlexCard embedded on Provider__c Lightning Record Page via App Builder. Read access to PNA__c and Fee_Schedule__c granted via ProviderNetworkSpecialist_PS Permission Set.
