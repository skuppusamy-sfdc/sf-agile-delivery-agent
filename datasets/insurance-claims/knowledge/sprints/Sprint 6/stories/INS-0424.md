## Description

As a Provider Network Specialist, I need a Provider 360 FlexCard on the Provider__c (Person Account) record page that consolidates credentialing status from Credentialing_Case__c, active PNA summary from Provider_Network_Agreement__c, and network tier badge so that I can assess a provider's complete network standing in a single view without navigating across multiple related lists.

## Acceptance Criteria

Given a Provider Network Specialist opens a Provider__c record,
When the record page loads,
Then the 'Provider_360_Card' FlexCard renders in the right-hand panel displaying: (a) Credentialing_Case__c.Status__c for the most recent open or completed Credentialing_Case__c linked via Credentialing_Case__c.Provider__c; (b) count of active Provider_Network_Agreement__c records where PNA.Provider__c = current Provider__c and PNA.Status__c = 'Active'; (c) a color-coded network tier badge sourced from Provider__c.Network_Tier__c (Tier 1 = green, Tier 2 = yellow, Tier 3 = red, Out-of-Network = grey).

Given a provider has no active Credentialing_Case__c,
When the FlexCard renders,
Then the credentialing section displays 'No Active Credentialing Record' with a 'Initiate Credentialing' quick action button.

Given the provider has more than one active PNA,
When the specialist clicks 'View Agreements',
Then a flyout list renders showing PNA.Agreement_Number__c, PNA.Effective_Date__c, PNA.Expiration_Date__c, and PNA.LOB__c for each active PNA.

-Given the FlexCard also shows Fee_Schedule__c line counts per PNA- (deprecated — Fee Schedule detail moved to separate Fee Schedule 360 component per design review 2024-11-14)

Given Provider__c.NPI__c is populated,
When the FlexCard renders,
Then an 'NPPES Verify' action button is visible that triggers a callout to the NPPES API and displays last verified date from Provider__c.NPPES_Last_Verified__c.

## Solution

Create OmniStudio FlexCard 'Provider_360_Card' with three DataRaptor Extract actions: DR_Extract_LatestCredentialingCase (SOQL ordered by CreatedDate DESC, limit 1), DR_Extract_ActivePNAs (filtered by Status__c = 'Active'), and a direct field reference to Provider__c.Network_Tier__c. Tier badge uses conditional styling via FlexCard State configuration. 'NPPES Verify' button calls an Apex action (NPPESVerificationService.cls) that performs a named credential-based HTTP callout and updates Provider__c.NPPES_Last_Verified__c. Deploy via OmniStudio deployment tool to Production sandbox.
