## Description

As a Claims Operations Manager, I need an Agentforce AI assistant embedded on the Claim__c record page that analyzes claim attributes and recommends a triage action (e.g., 'Route to SIU', 'Approve for Auto-Adjudication', 'Request Additional Documentation'), so that claims review time is reduced and high-risk claims are flagged before manual review cycles begin.

## Acceptance Criteria

Given a Claim__c record is opened with Claim__c.Status__c = 'In Review',
When the Agentforce panel is activated by the Claims Operations Manager,
Then the AI assistant analyzes Claim__c.Total_Billed_Amount__c, Claim__c.Line_of_Business__c, Claim__c.Provider__c history, and Claim_Line__c.CPT_Code__c patterns and returns a triage recommendation within 5 seconds.

Given the AI recommendation is 'Route to SIU',
When the Claims Operations Manager accepts the recommendation,
Then Claim__c.SIU_Flag__c is set to TRUE, Claim__c.Status__c is updated to 'SIU Review', and a Task is created assigned to the SIU queue with Task.Subject = 'AI-Flagged: SIU Review Required'.

Given the AI recommendation is 'Approve for Auto-Adjudication',
When the recommendation is accepted,
Then a Record-Triggered Flow initiates the auto-adjudication path and Claim__c.Auto_Adjudication_Flag__c = TRUE is stamped with Claim__c.AI_Recommendation_Date__c.

Given the AI recommendation is generated,
When it is displayed in the Agentforce panel,
Then the recommendation includes a confidence score (AI_Confidence_Score__c expressed as a percentage) and the top 3 contributing factors as plain-language text.

Given any AI triage recommendation is made,
When the user accepts or rejects it,
Then an AI_Audit_Log__c record is created capturing: AI_Audit_Log__c.Claim__c, AI_Audit_Log__c.Recommendation__c, AI_Audit_Log__c.User_Action__c ('Accepted'/'Rejected'), AI_Audit_Log__c.Timestamp__c for compliance reporting.

## Solution

Configure Agentforce Agent with custom Topic 'ClaimsTriage' and Actions mapped to Apex-invocable methods: 'GetClaimRiskScore', 'RouteToSIU', 'FlagAutoAdjudication'. Build Einstein Prompt Template 'ClaimTriagePrompt' referencing Claim__c merge fields. Implement AI_Audit_Log__c creation via After-action Apex invocable. Embed Agentforce LWC component 'agentforceSidebar' on Claim__c Lightning record page. Ensure all prompt data is scoped within org data boundary; no PII transmitted externally.
