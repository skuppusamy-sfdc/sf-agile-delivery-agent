## Description

As a Claims Operations Manager, I need the Agentforce Claims Copilot embedded in the Claim__c record page to answer natural language questions about applicable fee schedule rates for a given CPT code and service date, so that claims adjusters can quickly validate allowed amounts without switching context to run SOQL queries or navigate fee schedule records manually.

## Acceptance Criteria

Given an adjuster types a query such as 'What is the allowed amount for CPT 99213 for a Commercial LOB claim with a service date of 12/01/2024?' into the Agentforce Copilot panel on the Claim__c record page,
When the agent processes the query,
Then it returns the correct Allowed_Amount__c value from the matching Fee_Schedule__c record, cites the Fee_Schedule__c.Name as the source, and displays the Effective_Date__c and Expiration_Date__c.

Given the Copilot retrieves a rate,
When the response is rendered,
Then a 'Apply to Current Line' quick action button is shown that pre-populates Claim_Line__c.CPT_Code__c and Claim_Line__c.Allowed_Amount__c on the currently open line item.

Given the agent cannot find a matching fee schedule for the queried CPT and date,
When the response is returned,
Then the Copilot responds with 'No active rate found for CPT [code] on [date]. Would you like me to create a rate gap case?' and offers a one-click Case creation action.

Given the adjuster asks about a CPT code outside their LOB permission scope,
When the agent evaluates the query,
Then it returns 'Rate information for this LOB is restricted to your role' and logs the query attempt in an Agentforce_Query_Audit__c custom object.

## Solution

Configure an Agentforce Agent Action using a custom Apex Tool class 'FeeScheduleRateLookupTool' that accepts CPT code, service date, and LOB as parameters and returns a structured JSON response. Register the tool in Agentforce Agent Builder with a natural language description for the LLM routing layer. Use Einstein Copilot's record context to auto-inject the current Claim__c.Line_of_Business__c into the tool call. The 'Apply to Current Line' button is an LWC quick action embedded in the Copilot response card. Agentforce_Query_Audit__c records are created via a Platform Event to avoid governor limit issues in the synchronous Copilot call stack.
