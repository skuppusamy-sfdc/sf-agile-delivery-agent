## Description

As a Provider Network Specialist, I need a FlexCard embedded in the Provider 360 console page that displays all currently active Fee_Schedule__c line items applicable to a given provider so that I can quickly reference contracted rates during provider inquiries without navigating away from the provider record.

## Acceptance Criteria

Given a Provider__c (Person Account) record page is open in the console,
When the Fee Schedule FlexCard loads,
Then it must display all Fee_Schedule_Line__c records linked to the provider's active PNA where Fee_Schedule__c.Status__c = 'Published' and Fee_Schedule__c.Expiration_Date__c >= TODAY().

Given the FlexCard is rendered,
When data is displayed,
Then each row must include: CPT_Code__c, Description__c, Unit_Rate__c, Modifier__c, Effective_Date__c, and LOB__c.

Given more than 20 Fee_Schedule_Line__c records exist for a provider,
When the FlexCard loads,
Then pagination must be implemented with a default page size of 10 records and a 'Load More' action.

Given the Provider Network Specialist clicks a CPT code row,
When the row is selected,
Then a flyout panel must display the full Fee_Schedule__c parent record details including Network_Tier__c and Contract_Type__c.

-Given the FlexCard is visible to all internal users, no permission filtering is required.- (Deprecated: Permission Set Group 'Provider_Network_ReadOnly' must gate visibility per security review)

## Solution

Build an OmniStudio FlexCard 'ProviderFeeScheduleCard' using a DataRaptor Extract as the data source, querying Fee_Schedule_Line__c with parent Fee_Schedule__c fields via relationship SOQL. Embed the FlexCard on the Provider__c Lightning Record Page using a custom App Builder region. Implement a child FlexCard for the CPT detail flyout. Apply Conditional Visibility on the FlexCard using the Permission Set Group 'Provider_Network_ReadOnly' check via a custom lwc wrapper. Cache DataRaptor results for 5 minutes using OmniStudio caching configuration to reduce SOQL calls on high-traffic console sessions.
