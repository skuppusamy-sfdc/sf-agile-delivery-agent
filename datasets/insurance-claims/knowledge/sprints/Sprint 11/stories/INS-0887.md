## Description

As a Broker Relations Coordinator, I need the system to automatically send rate change notification emails to affiliated brokers when a Group_Service_Agreement__c (GSA) fee schedule is amended, so that brokers can update their clients' (employers') benefit materials and remain compliant with state notification requirements.

Brokers are stored as Contact records linked to the Agency__c account. The GSA is linked to the Group/Employer Account via Group_Service_Agreement__c.Account__c.

## Acceptance Criteria

Given a Group_Service_Agreement__c record where GSA_Status__c changes to 'Rate Amendment Executed' and Amendment_Effective_Date__c is set,
When the record is saved,
Then all active broker Contacts linked via Broker_GSA_Assignment__c junction object receive an email notification using the 'GSA_Rate_Change_Broker_Alert' email template within 1 hour.

Given the email notification is sent,
When the email is delivered,
Then a PCN record (Provider_Change_Notification__c) is created with PCN_Type__c = 'GSA Broker Rate Alert', Related_GSA__c populated, and PCN_Status__c = 'Delivered'.

Given a broker Contact with Email_Opt_Out__c = TRUE,
When the notification batch runs,
Then that Contact is excluded from the email send and a log entry is written to Notification_Exclusion_Log__c with Reason__c = 'Email Opt-Out'.

Given the notification email is sent,
When the broker clicks the 'View Rate Summary' link in the email,
Then they are directed to the Experience Cloud Broker Portal page displaying the GSA rate summary.

## Solution

1. Record-Triggered Flow on Group_Service_Agreement__c (after-save) detects GSA_Status__c = 'Rate Amendment Executed' and enqueues a Queueable Apex job (BrokerRateChangeNotificationJob).
2. Queueable Apex queries Broker_GSA_Assignment__c for active broker Contacts, respects Email_Opt_Out__c, and sends via Messaging.sendEmail using the 'GSA_Rate_Change_Broker_Alert' template.
3. After send, Apex creates PCN records and writes exclusion logs in a single DML transaction.
4. Experience Cloud Broker Portal page uses a Lightning Web Component to display GSA rate summary via an Apex controller querying Fee_Schedule__c records related to the GSA.
