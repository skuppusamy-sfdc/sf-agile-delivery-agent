# INS-0975: Build Agentforce Claims AI prompt to recommend fee schedule disputes for underpaid Claim Lines

## Description

As a Claims Operations Manager, I need an Agentforce AI action that analyzes Claim_Line__c records where Paid_Amount__c is less than Allowed_Amount__c by more than 5% and recommends whether a fee schedule dispute should be initiated, so that claims examiners can quickly identify underpaid claims and take action without manually cross-referencing fee schedule tables and EOB documents.

## Acceptance Criteria

Given a Claims Examiner opens a Claim__c record and invokes the Agentforce 'Analyze Underpaid Lines' action, When the AI processes the request, Then it returns a structured recommendation for each Claim_Line__c where (Allowed_Amount__c - Paid_Amount__c) / Allowed_Amount__c > 0.05, including: Claim_Line__c ID, CPT_Code__c, variance amount, applied Fee_Schedule__c ID, and a recommended action of 'Initiate Dispute' or 'Review EOB'.

Given the Agentforce response is displayed, When the examiner selects 'Initiate Dispute' on a recommended line, Then a Case is created with RecordType = 'Fee Schedule Dispute', linked to the Claim__c and Claim_Line__c, with Subject auto-populated as 'Fee Schedule Dispute - [CPT Code] - [Claim Number]'.

Given the AI prompt template is configured, When the prompt references claim data, Then it uses grounding data from Claim_Line__c.Applied_Fee_Schedule__r.New_Rate__c, Claim_Line__c.Paid_Amount__c, and EOB__c.Remittance_Remark_Code__c to provide context-aware recommendations.

Given the Agentforce action is invoked on a Claim__c with no underpaid lines, When the analysis completes, Then the response states 'No underpaid claim lines detected. All lines are within the 5% variance threshold.'

## Solution

Configure an Agentforce Agent Action of type 'Prompt Template' using Einstein Generative AI in Salesforce. Build a Flex Prompt Template 'Underpaid_Claim_Line_Analysis_PT' that uses a DataRaptor-sourced merge field block to inject Claim_Line__c records (filtered by variance > 5%) into the prompt context window. Ground the prompt with Fee_Schedule__c and EOB__c related data via a custom Apex invocable that returns a structured JSON context payload. Define the Agent Topic 'Claims Fee Schedule Analysis' with the action bound to the Claim__c object. The 'Initiate Dispute' quick action is an Apex-backed invocable triggered from the Agentforce response UI using a Flow-based confirmation dialog. Ensure all AI responses are logged to a custom AI_Interaction_Log__c object for compliance auditing.
