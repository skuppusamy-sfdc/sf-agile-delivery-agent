## Description

As a Claims Operations Manager, I need newly created Claim__c records to be automatically routed to the correct adjuster queue based on the claim's LOB and the insured's state of residence so that claims are worked by the appropriately licensed and specialized team without manual triage delay.

Extends INS-0268 (FNOL intake creates the Claim__c record that triggers this assignment logic).

## Acceptance Criteria

Given a Claim__c record is created with Claim__c.Status__c = 'FNOL Received',
When the assignment rule evaluates,
Then if Claim__c.LOB__c = 'Medical' AND Claim__c.Insured_State__c IN ('CA','NY','TX') the record is assigned to Queue: 'Medical_Claims_West_East'.

Given Claim__c.LOB__c = 'Auto',
When the assignment rule evaluates,
Then the record is assigned to Queue: 'Auto_Claims_<State>' where <State> matches Claim__c.Insured_State__c.

Given no matching queue rule exists for the LOB/State combination,
When the fallback rule is evaluated,
Then the claim is assigned to Queue: 'Claims_General_Intake' and Claim__c.Assignment_Flag__c = 'Unmatched — Manual Review'.

Given assignment completes,
When the Claim__c owner changes to a Queue,
Then an in-app notification is sent to all Queue members via Custom Notification Type 'New_Claim_Assigned'.

Given a TPA-managed policy (Policy__c.TPA_Managed__c = true),
When the assignment rule runs,
Then the claim bypasses internal queues and Claim__c.TPA_Routing_Flag__c is set to true with owner remaining the submitting user pending TPA handoff.

## Solution

Implement Salesforce Assignment Rules on Claim__c object with criteria-based rule entries ordered by LOB and State. Configure a Record-Triggered Flow (After Insert on Claim__c, filter: Status__c = 'FNOL Received') to invoke the assignment rule via the 'Assign Using Active Assignment Rule' flow action. For TPA bypass logic, add a Decision element in the flow checking Policy__c.TPA_Managed__c before triggering assignment. Custom Notification sent via the Send Custom Notification flow action. Create Permission Set to grant Claims Operations Manager visibility to all queues.
