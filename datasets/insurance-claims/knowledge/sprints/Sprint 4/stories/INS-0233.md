## Description

As a Claims Operations Manager, I need the system to automatically generate and deliver an Explanation of Benefits (EOB) document to the policyholder when a claim reaches 'Adjudicated' or 'Partially Adjudicated' status so that claimants receive timely, compliant benefit summaries as required by state regulations.

## Acceptance Criteria

Given a Claim__c.Status__c transitions to 'Adjudicated' or 'Partially Adjudicated',
When the EOB generation process triggers,
Then a new Explanation_of_Benefits__c record is created with EOB__c.Claim__c populated, EOB__c.Generated_Date__c = today, and EOB__c.Status__c = 'Draft'.

Given the EOB__c record is in 'Draft' status,
When the Conga CLM merge process runs,
Then the EOB PDF is generated using the 'EOB_Standard_Template_v2' Conga template, merging Claim__c, Claim_Line__c, and Policy__c field data.

Given the EOB PDF is successfully generated,
When the document is attached to the EOB__c record,
Then EOB__c.Status__c = 'Generated' and the PDF is stored as a Salesforce File linked to both EOB__c and Claim__c.

Given EOB__c.Status__c = 'Generated' and Policy__c.Preferred_Delivery__c = 'Email',
When the delivery step executes,
Then an email is sent to Policy__c.Policyholder_Email__c using the 'EOB_Delivery' email template and EOB__c.Delivery_Status__c = 'Sent'.

Given Policy__c.Preferred_Delivery__c = 'Portal',
When the delivery step executes,
Then EOB__c.Portal_Published__c = true and the document is made available in the Policyholder Experience portal.

Given the EOB generation fails due to a Conga error,
When the exception is caught,
Then EOB__c.Status__c = 'Error', an error message is written to EOB__c.Error_Details__c, and a Case is auto-created and assigned to the Claims Operations queue.

## Solution

Use a Record-Triggered Flow on Claim__c (After Update, Status__c IN ['Adjudicated','Partially Adjudicated']) to create the EOB__c record and invoke a Conga CLM action via the Conga Batch API through an Apex callout class (CongaEOBService.cls). Handle async callback from Conga to update EOB__c.Status__c via a named credential and REST endpoint. For email delivery, use Salesforce Email Alerts configured in the Flow. For portal delivery, set EOB__c.Portal_Published__c via a Flow update. Implement error handling with a custom Platform Event (EOB_Error__e) to trigger Case creation asynchronously.
