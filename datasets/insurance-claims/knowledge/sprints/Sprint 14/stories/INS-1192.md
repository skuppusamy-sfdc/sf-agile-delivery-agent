## Description

As a Claims Operations Manager, I need the Agentforce AI agent to automatically generate a draft Explanation_of_Benefits__c (EOB) record and associated PDF document when a Claim__c reaches 'Adjudicated' status so that the manual EOB authoring step is eliminated, cutting EOB turnaround time from 48 hours to under 2 hours. Extends INS-1178 (EOB data model enhancements).

## Acceptance Criteria

Given Claim__c.Status__c transitions to 'Adjudicated' and all related Claim_Line__c records have Claim_Line__c.Adjudication_Status__c = 'Processed',
When the Record-Triggered Flow fires on Claim__c update,
Then a new Explanation_of_Benefits__c record must be created with EOB__c.Claim__c populated, EOB__c.Generation_Method__c = 'AI Draft', and EOB__c.Status__c = 'Pending Review'.

Given the EOB draft is created,
When the Agentforce document generation action runs,
Then EOB__c.Draft_Document_URL__c must contain a valid Salesforce Files (ContentDocument) link to a generated PDF within 5 minutes.

Given the generated EOB PDF,
When a Claims Adjudicator reviews the draft,
Then all Claim_Line__c fields including Claim_Line__c.Billed_Amount__c, Claim_Line__c.Allowed_Amount__c, Claim_Line__c.Patient_Responsibility__c, and Claim_Line__c.CPT_Code__c must be accurately reflected in the document.

Given EOB__c.Status__c = 'Pending Review',
When a Claims Adjudicator approves the EOB via the Approval Process,
Then EOB__c.Status__c must transition to 'Approved' and EOB__c.Approved_By__c and EOB__c.Approval_Date__c must be stamped.

Given the EOB is approved,
When EOB__c.Status__c = 'Approved',
Then a DocuSign envelope must be triggered automatically to send the EOB to the policyholder email stored on Policy__c.Policyholder_Email__c.

## Solution

Record-Triggered Flow on Claim__c (After Update, Status__c changed to 'Adjudicated') creates the Explanation_of_Benefits__c record and invokes an Agentforce Agent Action. The action uses a Conga CLM template configured for EOB generation, pulling Claim_Line__c data via a DataRaptor Extract. The generated PDF is stored as a Salesforce File linked to the EOB record. An Approval Process on Explanation_of_Benefits__c manages the review step. Upon final approval, a Flow invokes the DocuSign for Salesforce API via a named credential to dispatch the member-facing EOB. AI generation prompts are managed in the Agentforce Prompt Builder using a custom Claim EOB prompt template.
