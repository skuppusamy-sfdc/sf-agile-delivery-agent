## Description

As a Claims Operations Manager, I need the Agentforce AI agent to automatically triage incoming FNOL submissions by Line of Business (LOB) and injury severity score so that claims are routed to the correct adjuster queue within seconds of intake, reducing manual triage time by at least 60%.

## Acceptance Criteria

Given a new Claim__c record is created with Claim__c.Stage__c = 'FNOL' and Claim__c.LOB__c is populated,
When the Agentforce triage topic fires,
Then Claim__c.Severity_Score__c is populated (range 1–5) based on ICD-10 diagnosis codes from associated Claim_Line__c records within 10 seconds.

Given Claim__c.Severity_Score__c >= 4,
When triage completes,
Then Claim__c.Owner is reassigned to the 'Complex Claims' queue and Claim__c.Priority__c = 'High'.

Given Claim__c.LOB__c = 'Workers Compensation',
When AI triage runs,
Then Claim__c.Assigned_Adjuster_Queue__c = 'WC Complex' if severity >= 4, else 'WC Standard'.

Given Claim__c.LOB__c = 'General Liability',
When triage completes,
Then a Task is auto-created with Subject = 'FNOL Review – GL' assigned to the GL adjuster queue owner.

-Given Claim__c.Fraud_Indicator__c = true, route directly to SIU queue- (deprecated: SIU routing handled by INS-1198)

Given the Agentforce topic fails to score a claim due to missing ICD-10 data,
When triage is attempted,
Then Claim__c.Triage_Status__c = 'Manual Review Required' and an alert email is sent to the Claims Operations Manager.

## Solution

Configure an Agentforce Agent Topic 'FNOL Claim Triage' with a custom action invoking an Apex invocable method (ClaimTriageAction.cls) that queries Claim_Line__c records for ICD-10 codes and calculates a severity score using a Custom Metadata-driven scoring matrix (Claim_Severity_Rule__mdt). A Record-Triggered Flow on Claim__c (after insert, Stage__c = 'FNOL') fires a Platform Event (FNOL_Triage_Event__e) consumed by the Agentforce topic. Queue assignment is handled via Apex with OmniStudio DataRaptor for field mapping. Einstein Next Best Action surfaces triage recommendations on the Claims Console FlexCard.
