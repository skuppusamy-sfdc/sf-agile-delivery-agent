## Description

As a Provider Relations Specialist, I need a consolidated 'Contract & Rates' panel on the Provider__c (Person Account) record page so that I can view active Provider_Network_Agreement__c terms and associated Fee_Schedule__c rates without navigating to multiple related lists.

The panel should display the most recent active PNA, key contract dates, network tier, and a sample of top-20 CPT code rates from the linked Fee_Schedule__c.

## Acceptance Criteria

Given a Provider Relations Specialist views a Provider__c record,
When the 'Contract & Rates' FlexCard loads,
Then it displays the active Provider_Network_Agreement__c where PNA__c.Status__c = 'Active' AND PNA__c.Provider__c = current Provider__c.Id, showing PNA__c.Contract_Start_Date__c, PNA__c.Contract_End_Date__c, PNA__c.Network_Tier__c, and PNA__c.Reimbursement_Model__c.

Given the active PNA has a linked Fee_Schedule__c,
When the panel renders the rates section,
Then a child DataRaptor Extract retrieves up to 20 Fee_Schedule__c line items (Fee_Schedule_Line__c) ordered by Fee_Schedule_Line__c.CPT_Code__c ASC, displaying CPT_Code__c, Fee_Schedule_Line__c.Allowed_Amount__c, and Fee_Schedule_Line__c.Effective_Date__c.

Given no active PNA exists for the provider,
When the FlexCard loads,
Then a message reads 'No active network agreement on file. Contact Provider Network Contracting.' with a quick action button to create a new PNA record.

Given PNA__c.Contract_End_Date__c is within 60 days of TODAY(),
When the panel displays contract dates,
Then the expiration date field is rendered in amber with a warning badge 'Renewal Due'.

-Given the panel includes a 'Download Fee Schedule' button linking to a Conga CLM generated PDF-
-Deferred to Phase 3 pending Conga CLM upgrade-

## Solution

Build OmniStudio FlexCard (ProviderContractRates_FC) with two DataRaptor Extract actions: one targeting Provider_Network_Agreement__c filtered by Status__c = 'Active' and Provider__c = recordId, and a second targeting Fee_Schedule_Line__c with a SOQL LIMIT 20 ordered by CPT_Code__c. Conditional rendering in FlexCard for amber styling when days-until-expiry < 60 using a calculated field. Deploy FlexCard to Provider__c record page via App Builder. Permission Set 'Provider_Relations_Standard' grants read access to PNA__c and Fee_Schedule_Line__c.
