## Description

As a Broker Relations Coordinator, I need a real-time Year-To-Date (YTD) commission earnings dashboard within the Agent Portal so that agents and brokers can self-serve their commission summaries without contacting the back office, reducing inbound inquiry volume and improving agent satisfaction. The dashboard should aggregate commissions across all active LOBs (Commercial, Individual, Medicare Supplement) and display earned vs. pending commission amounts tied to Policy__c records.

## Acceptance Criteria

Given an authenticated agent accesses the Agent Portal Commission Dashboard,
When the page loads,
Then a FlexCard summary panel displays Commission_Summary__c.YTD_Earned__c, Commission_Summary__c.YTD_Pending__c, and Commission_Summary__c.Last_Payment_Date__c for the logged-in agent's NPN.

Given the agent selects a LOB filter (Commercial, Individual, Medicare Supplement),
When the filter is applied,
Then the dashboard re-queries Commission_Line__c records where Commission_Line__c.LOB__c matches the selected value and refreshes all summary totals within 2 seconds.

Given a commission payment has been processed and Payment__c.Status__c = 'Disbursed',
When the agent views the Commission History sub-tab,
Then each row displays Payment__c.Amount__c, Payment__c.Disbursement_Date__c, Policy__c.Policy_Number__c, and Payment__c.Commission_Type__c (New Business, Renewal, Override).

Given an agent has pending commissions where Commission_Line__c.Status__c = 'Pending Approval',
When the agent hovers over the Pending amount,
Then a tooltip displays the count of pending Commission_Line__c records and the earliest expected approval date from Commission_Line__c.Expected_Approval_Date__c.

-Given the agent can export the full commission report as CSV directly from the dashboard-
-When the export button is clicked-
-Then a CSV file is downloaded containing all Commission_Line__c fields for the current filter selection-
(Deferred to INS-1005 — export functionality moved to separate story)

Given the agent's profile does not have the 'Commission_View' Permission Set Group assigned,
When they navigate to the Commission Dashboard tab,
Then the tab is hidden and a 'Contact your administrator' message is displayed.

## Solution

Build the Commission Dashboard as an LWC-based FlexCard (OmniStudio) embedded in the Agent Portal community. A DataRaptor Extract retrieves Commission_Summary__c and Commission_Line__c records filtered by the logged-in user's Agent__c lookup (matched via NPN on User.NPN__c). A Platform Event (Commission_Updated__e) is published by an Apex Trigger on Payment__c (after insert/update where Status__c = 'Disbursed') to push real-time refresh signals to the FlexCard via an LWC event listener. LOB filter state is managed via OmniScript session variables. Access controlled via Permission Set Group 'Agent_Commission_Viewer'. Community sharing rules ensure agents only see their own Commission_Line__c records (Sharing Rule: Owner = Current User's Agent__c).
