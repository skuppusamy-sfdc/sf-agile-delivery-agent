## Description

As a Broker Relations Coordinator, I need a real-time commission earnings dashboard within the Agent Portal so that brokers and agents can self-serve their commission summaries by LOB, policy, and payment period without contacting the back-office team.

This story extends INS-1036 (Agent Portal base layout) to add a Commission Summary FlexCard and supporting DataRaptors for commission aggregation.

## Acceptance Criteria

Given an authenticated agent user with the 'Agent_Portal_User' Permission Set Group assigned,
When they navigate to the Commission Dashboard tab in the Agent Portal,
Then a FlexCard renders displaying total YTD commissions aggregated from Commission__c.Amount__c grouped by Commission__c.LOB__c.

Given commission records exist on Commission__c with Commission__c.Status__c = 'Paid',
When the dashboard loads,
Then only records where Commission__c.Payment_Period__c falls within the current calendar year are displayed in the YTD total.

Given the agent selects a specific Policy__c from the policy picklist filter,
When the filter is applied,
Then the FlexCard refreshes to show Commission__c records linked to that Policy__c.Id only.

Given Commission__c.Status__c = 'Pending',
When the agent views the Pending Commissions panel,
Then the estimated payout date from Commission__c.Estimated_Payment_Date__c is displayed alongside the pending amount.

-Given a CSV export button is available, When clicked, Then all commission rows export to CSV.- (Deprecated: export functionality moved to INS-1081)

Given the dashboard is accessed on a mobile viewport (< 768px),
When the FlexCard renders,
Then all commission summary cards stack vertically and no horizontal scroll is required.

## Solution

Build an OmniStudio FlexCard ('Agent_Commission_Dashboard') with two child DataRaptors: (1) 'DR_Get_Agent_Commissions' — a Turbo Extract DataRaptor querying Commission__c filtered by OwnerId = :currentUserId and Status__c IN ('Paid','Pending'); (2) 'DR_Aggregate_Commission_By_LOB' — a Transform DataRaptor performing group-by aggregation on LOB__c. The FlexCard is surfaced via an LWC wrapper ('agentCommissionDashboard') embedded in the Agent Portal Experience Cloud page. A Platform Event (Commission_Updated__e) triggers a real-time refresh via a Record-Triggered Flow on Commission__c on after-update events. Permission access controlled via 'Agent_Portal_User' Permission Set Group.
