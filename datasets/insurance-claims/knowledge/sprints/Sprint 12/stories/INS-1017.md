## Description

As a Broker Relations Coordinator, I need agents to be able to generate a formatted Commission Statement PDF directly from the Agent Portal Commission Dashboard so that agents have an official record of their earnings for reconciliation, tax preparation, and dispute resolution purposes.

Note: This story was split out from INS-1011 (Commission Dashboard) where the PDF export requirement was deprecated from that story's scope. This story delivers that capability as a standalone feature.

## Acceptance Criteria

Given an agent is viewing the Commission Dashboard and has at least one Payment__c record with Commission_Status__c = 'Paid' in the selected period,
When the agent clicks 'Download Commission Statement',
Then a Conga CLM merge is triggered using template 'Commission_Statement_v2' and a PDF is generated and downloaded to the user's browser within 15 seconds.

Given the PDF is generated,
When the document is opened,
Then it contains: agent name, NPN (Agent__c.NPN__c), statement period, itemized list of Payment__c records (date, policy number, LOB, premium, rate, commission amount), and a total row.

Given the agent selects a custom date range filter before clicking download,
When the Conga merge executes,
Then only Payment__c records where Payment__c.Payment_Date__c falls within the selected range are included in the statement.

Given the generated PDF,
When it is saved,
Then it is automatically attached to the Agent__c record as a ContentDocument linked via ContentDocumentLink with a file name format 'Commission_Statement_[AgentNPN]_[YYYY-MM].pdf'.

## Solution

Add 'Download Commission Statement' action button to the Commission Dashboard FlexCard. Button triggers an Apex @AuraEnabled method 'CommissionStatementService.generateStatement()' that assembles the Payment__c data set and calls the Conga CLM API (via Named Credential 'Conga_CLM') with template ID stored in Custom Metadata 'Conga_Config__mdt'. Conga response (base64 PDF) decoded in Apex and saved as ContentVersion linked to Agent__c via ContentDocumentLink. LWC handles download via NavigationMixin once ContentDocument ID is returned. Permission gated by custom permission 'Download_Commission_Statement' included in Permission Set Group 'Agent_Portal_Commission_View'.
