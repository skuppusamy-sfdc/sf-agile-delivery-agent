## Description

As a Broker Relations Coordinator, I need a recruitment pipeline tracker within the Agent Portal so that I can monitor the onboarding status of prospective brokers from initial application through licensing verification, appointment, and first policy bind — reducing manual follow-up and improving time-to-active-agent metrics.

Prospective broker records are stored as Lead objects with a custom record type 'Broker_Prospect'. Once qualified, they convert to Contact + Account (Agency) with a linked Agent__c record. Each stage must map to a defined onboarding milestone.

## Acceptance Criteria

Given a Broker_Prospect Lead is created with RecordType = 'Broker_Prospect',
When the record is saved,
Then a Record-Triggered Flow sets Lead.Onboarding_Stage__c = 'Application Received' and creates a Task for the owning Broker Relations Coordinator due in 2 business days.

Given the pipeline tracker kanban view is loaded in the Agent Portal,
When the coordinator filters by 'Region' picklist,
Then only Lead records where Lead.Broker_Region__c matches the selected value are displayed.

Given a Lead reaches Onboarding_Stage__c = 'License Verification',
When the stage is saved,
Then a CAQH integration callout (via Named Credential 'CAQH_API') is triggered to retrieve NPI and license status, populating Lead.NPI__c and Lead.License_Status__c.

Given license verification returns a status of 'Active' from CAQH,
When the response is processed,
Then Lead.Onboarding_Stage__c auto-advances to 'Appointment Pending' and a DocuSign envelope is sent via the DocuSign API for the broker appointment agreement.

Given the DocuSign envelope status = 'Completed',
When the webhook callback is received,
Then the Lead converts to Contact + Agent__c with Agent__c.Status__c = 'Active' and Agent__c.Appointment_Date__c = today.

## Solution

OmniScript 'Broker_Onboarding_Wizard' guides coordinators through stage transitions with embedded DataRaptor Load steps. Record-Triggered Flow on Lead handles stage-based task creation. Apex Callout class 'CAQHIntegrationService' invokes CAQH REST API using Named Credential; response mapped via DataRaptor Transform. DocuSign envelope creation via Apex 'DocuSignBrokerAppointmentService'. Platform Event 'DocuSign_Callback__e' triggers a second Record-Triggered Flow for Lead conversion. FlexCard 'Broker_Pipeline_Kanban' renders drag-and-drop stage columns using OmniStudio.
