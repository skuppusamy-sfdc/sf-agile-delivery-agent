## Description

As a Provider Relations Specialist, I need a consolidated Provider Termination 360 FlexCard on the Provider__c (Person Account) record page that surfaces active Claim__c records, open Provider_Network_Agreement__c entries, Credentialing_Case__c status, and any in-flight Network_Termination_Case__c records, so that I have a single-pane-of-glass view before finalizing a provider exit and can identify downstream claim impacts.

## Acceptance Criteria

Given a Provider__c record with at least one active Claim__c (Claim__c.Status__c NOT IN ('Closed','Denied','Void'))
When the FlexCard is rendered on the provider record page
Then a 'Active Claims' tile displays count and a datatable showing Claim__c.Name, Claim__c.Total_Billed_Amount__c, Claim__c.LOB__c, and Claim__c.Adjudication_Status__c

Given a Provider__c record
When the FlexCard loads
Then all Provider_Network_Agreement__c records where PNA.Provider__c = current provider and PNA.Status__c = 'Active' are listed with columns: PNA.Network_Name__c, PNA.Contract_End_Date__c, PNA.LOB__c, and PNA.Renewal_Status__c

Given a Network_Termination_Case__c exists for the provider
When the FlexCard renders
Then a red banner displays 'Termination In Progress' with Network_Termination_Case__c.Status__c and Effective_Termination_Date__c visible

Given a Credentialing_Case__c linked to the provider
When the FlexCard loads
Then Credentialing_Case__c.CAQH_Status__c, Credentialing_Case__c.License_Expiry_Date__c, and Credentialing_Case__c.Board_Certification__c are displayed in a 'Credentialing Summary' section

-Display open Payment__c disbursements in FlexCard- (deprecated: moved to standalone Payment 360 card per INS-0791)

Given the FlexCard is displayed
When a user with Profile 'Read-Only Provider Ops' views the card
Then all action buttons (e.g., 'Initiate Termination', 'Send PCN') are hidden and only read-only data tiles are visible

## Solution

Build OmniStudio FlexCard 'ProviderTermination360' with multiple DataRaptor Extract sources: one for active Claim__c, one for PNA records, one for Credentialing_Case__c, and one for Network_Termination_Case__c. Use conditional visibility states on the FlexCard to toggle the red termination banner. Embed the FlexCard on the Provider__c Lightning Record Page via App Builder. Apply OmniStudio Action to launch the Termination OmniScript (INS-0802) directly from the card. Leverage Permission Set Group 'Provider_Network_Ops' for action visibility and a custom Apex DataRaptor Turbo Extract for performant SOQL on large claim volumes.
