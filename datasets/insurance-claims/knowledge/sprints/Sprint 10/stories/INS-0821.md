## Description

As a Compliance Officer, I need the system to automatically generate and distribute a Provider Change Notification (PCN) letter to the affected provider via DocuSign when a Network_Termination_Case__c reaches Status__c = 'Pending Termination' so that we satisfy state-mandated notice requirements (minimum 60-day notice) and maintain an auditable delivery record on the provider record.

## Acceptance Criteria

Given a Network_Termination_Case__c where NTC.Status__c transitions to 'Pending Termination'
When the Record-Triggered Flow fires (after save, on update)
Then a Provider_Change_Notification__c record is created with:
  - PCN.Provider__c = NTC.Provider__c
  - PCN.NTC__c = NTC.Id
  - PCN.Notification_Type__c = 'Termination Notice'
  - PCN.Notice_Date__c = TODAY
  - PCN.Required_Notice_Days__c = 60
  - PCN.Deadline_Date__c = TODAY + 60
  - PCN.Status__c = 'Pending Delivery'

Given the PCN record is created
When the Conga CLM merge is triggered
Then a PDF letter is generated using template 'PCN_Termination_v2' and attached to the PCN record as a ContentVersion with ContentVersion.Title = 'PCN_[NPI]_[YYYYMMDD].pdf'

Given the PDF is generated
When the DocuSign envelope is created via the DocuSign for Salesforce integration
Then the envelope is sent to Provider__c.PersonEmail with:
  - Envelope Subject = 'Important: Network Termination Notice - Action Required'
  - Signing role = 'Provider Acknowledgment' (signature required)
  - PCN.DocuSign_Envelope_Id__c populated with the returned envelope ID

Given the provider signs the DocuSign envelope
When the DocuSign Connect callback fires
Then PCN.Status__c = 'Delivered - Acknowledged', PCN.Acknowledged_Date__c = envelope completed date, and the signed PDF is attached to the NTC record

Given the provider does NOT sign within 14 days
When the scheduled reminder flow runs
Then a follow-up DocuSign reminder is sent and PCN.Reminder_Sent__c = TRUE is set

Given NTC.Termination_Reason__c = 'Voluntary Withdrawal'
When the PCN is generated
Then the letter template used is 'PCN_Voluntary_Exit_v1' instead of 'PCN_Termination_v2'

## Solution

Build a Record-Triggered Flow on Network_Termination_Case__c triggered on Status__c change to 'Pending Termination'. Create PCN record via Create Records element. Invoke an Apex Action 'CongaMergeService.generatePCN()' passing template name (determined by Termination_Reason__c via Decision element) and record IDs. On successful PDF generation, invoke 'DocuSignEnvelopeService.sendEnvelope()' Apex Action. Store envelope ID on PCN. Configure DocuSign Connect webhook to call a custom REST endpoint that updates PCN via a Platform Event 'DocuSign_Callback__e'. Build a separate Scheduled Flow for 14-day reminder logic querying PCN records where Status__c = 'Pending Delivery' and Notice_Date__c <= TODAY - 14 and Reminder_Sent__c = FALSE.
