## Description

As a Compliance Officer, I need a Salesforce Lightning dashboard that tracks provider termination notice compliance against the state-mandated 60-day notice rule, segmented by state and Line of Business, so that I can proactively identify and remediate any PCN delivery gaps before regulatory reporting deadlines. This story depends on PCN generation being operational per INS-0821.

## Acceptance Criteria

Given Provider_Change_Notification__c and Network_Termination_Case__c records exist in the system
When the Compliance Officer opens the 'Provider Termination Compliance' dashboard
Then the dashboard displays the following components:
  1. KPI Tile: Total open Network_Termination_Case__c records by NTC.Status__c
  2. Bar Chart: PCN records grouped by PCN.Status__c (Pending Delivery, Delivered - Acknowledged, Overdue)
  3. Table Report: PCN records where PCN.Deadline_Date__c < TODAY AND PCN.Status__c != 'Delivered - Acknowledged' (at-risk notices)
  4. Pie Chart: Termination cases by NTC.LOB__c
  5. Stacked Bar: PCN compliance rate by Provider__c.BillingState (state segmentation)

Given the 'At-Risk Notices' table report
When a PCN record shows PCN.Deadline_Date__c within 7 days of TODAY
Then the row is highlighted in amber; records past PCN.Deadline_Date__c are highlighted in red (conditional formatting)

Given the dashboard is viewed
When the Compliance Officer applies the 'State' filter
Then all dashboard components filter to show only records where Provider__c.BillingState = selected state

Given the dashboard data
When the Compliance Officer clicks 'Export to CSV' on the At-Risk Notices table
Then a CSV export is available containing: Provider NPI, TIN, PCN.Notice_Date__c, PCN.Deadline_Date__c, PCN.Status__c, NTC.LOB__c, Provider State

-Given the dashboard auto-refreshes every 5 minutes-
-Auto-refresh requirement removed per Compliance Officer feedback; manual refresh is sufficient-

## Solution

Create two custom Report Types in Salesforce: (1) 'NTC with PCN' joining Network_Termination_Case__c → Provider_Change_Notification__c → Provider__c; (2) 'PCN At-Risk' filtered report with PCN.Deadline_Date__c filter. Build 5 reports using these report types with appropriate groupings, filters, and summary formulas for compliance rate calculation. Assemble into a Lightning Dashboard 'Provider Termination Compliance Dashboard' with a 3-column layout. Implement conditional row highlighting using Report Chart formatting and Row-Level Formula fields on the PCN object (PCN.Days_Until_Deadline__c = PCN.Deadline_Date__c - TODAY as a formula field). Add a Dashboard Filter component on Provider__c.BillingState. Share dashboard via Folder permissions to 'Compliance_Officers' public group. Schedule daily email subscription to Compliance Officer distribution list.
