## Description

As a Provider Relations Specialist, I need a consolidated Provider Termination 360 FlexCard embedded on the Provider__c (Person Account) record page that surfaces active PNAs, in-flight claims, open Network_Termination_Cases, and the projected termination timeline, so that I can manage the full exit process from a single screen without navigating across multiple objects.

## Acceptance Criteria

Given a Provider__c record with one or more associated Provider_Network_Agreement__c records
When the Provider Termination 360 FlexCard loads
Then the FlexCard displays:
  - All PNA records where PNA.Status__c IN ('Active', 'Pending Termination') with fields: PNA.Name, PNA.LOB__c, PNA.End_Date__c, PNA.Termination_Type__c
  - Count of open Claim__c records where Claim.Rendering_Provider__c = Provider__c.Id AND Claim.Status__c NOT IN ('Paid', 'Denied', 'Void')
  - All Network_Termination_Case__c records linked to the provider
  - A visual timeline component showing key termination milestone dates

Given no active PNAs exist for the provider
When the FlexCard loads
Then a contextual message 'No active network agreements found' is displayed in the PNA panel

Given Claim__c.Status__c = 'In Adjudication' and Claim.Rendering_Provider__c = current Provider__c
When the Claims panel renders
Then each claim row displays Claim__c.Name, Claim__c.Total_Billed_Amount__c, Claim__c.Date_of_Service__c, and a hyperlink to the Claim record

Given the user clicks 'Initiate Termination' action on a PNA row
When the action fires
Then an OmniScript (Provider_Termination_Initiation) launches in a modal pre-populated with PNA.Provider__c and PNA.Id

## Solution

Build OmniStudio FlexCard 'Provider_Termination_360' with three child cards: PNA_Summary_Card, Open_Claims_Card, and Termination_Timeline_Card. DataRaptors (DR_Get_Provider_PNAs, DR_Get_Provider_Open_Claims, DR_Get_NTC_Records) retrieve data via SOQL with provider Id as input key. Timeline component uses a custom LWC (providerTerminationTimeline) embedded in the FlexCard via LWC Action. The 'Initiate Termination' flyout action invokes OmniScript 'Provider_Termination_Initiation_OS'. FlexCard deployed to Provider__c Lightning record page via App Builder. Visibility controlled by Permission Set Group 'Provider_Relations_PSG'.
