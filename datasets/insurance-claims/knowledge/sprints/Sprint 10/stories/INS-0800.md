## Description

As a Provider Network Specialist, I need the system to automatically create a Network_Termination_Case__c record when a Provider_Network_Agreement__c (PNA) reaches its Contract_End_Date__c with no active renewal, so that provider exit workflows are initiated without manual intervention and network compliance timelines are met.

## Acceptance Criteria

Given a Provider_Network_Agreement__c record where Contract_End_Date__c equals T-60 days and Renewal_Status__c is NOT 'In Progress' or 'Executed'
When the nightly scheduled batch job runs
Then a Network_Termination_Case__c is created with Status__c = 'Pending Review', Termination_Reason__c = 'Contract Expiration', and linked via PNA_Lookup__c to the originating PNA

Given the Network_Termination_Case__c is created
When the record is saved
Then a Provider_Change_Notification__c (PCN) record is auto-generated with Notification_Type__c = 'Termination Notice', Effective_Date__c = PNA.Contract_End_Date__c, and Status__c = 'Draft'

Given the PCN record is in 'Draft' status
When the Provider Relations Specialist reviews and submits
Then PCN.Status__c transitions to 'Sent' and a DocuSign envelope is dispatched to the provider's NPI-registered email on file

Given a PNA where Renewal_Status__c = 'Executed' exists
When the batch job evaluates the same Contract_End_Date__c threshold
Then NO Network_Termination_Case__c is created and the PNA is bypassed with a log entry in the batch job's Custom_Log__c object

-Given termination is triggered, auto-terminate all child Fee_Schedule__c records immediately- (deprecated: fee schedule sunset handled in INS-0803)

Given a Network_Termination_Case__c is created
When Status__c = 'Pending Review'
Then the assigned Provider Relations Specialist receives a Salesforce Bell Notification and a Task is created with Subject = 'Review Termination Case – Action Required' due within 5 business days

## Solution

Implement a Scheduled Apex class (ProviderTerminationBatchJob) running nightly at 02:00 UTC that queries PNA records meeting termination criteria. Use a Record-Triggered Flow on Network_Termination_Case__c (after insert) to auto-create the linked PCN and dispatch the DocuSign envelope via the DocuSign for Salesforce API. Custom_Log__c records written via a separate Apex helper class. Bell notifications delivered through Salesforce Notification Builder. Permission Set Group 'Provider_Network_Ops' grants write access to Network_Termination_Case__c and PCN objects. Platform Event (TerminationInitiated__e) published on case creation for downstream consumers.
