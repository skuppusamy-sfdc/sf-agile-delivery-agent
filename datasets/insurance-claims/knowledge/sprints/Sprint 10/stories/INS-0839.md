## Description

As a Claims Operations Manager and Compliance Officer, I need a Salesforce CRM Analytics (formerly Tableau CRM) dashboard surfacing key Provider Termination KPIs — including termination volumes by type, 90-day notice compliance rates, average time-to-confirm, and total claims value suspended per termination event — so that leadership can monitor network exit health and proactively identify compliance risks at a glance.

## Acceptance Criteria

Given the dashboard is deployed to the 'Provider Network Ops' app,
When a Compliance Officer or Claims Operations Manager with the 'Network_Reporting_Viewer' Permission Set opens it,
Then all widgets load within 5 seconds and reflect data as of the most recent nightly data sync.

Given the 'Termination Volume by Type' bar chart widget,
When the user applies the date range filter (default: rolling 90 days),
Then the chart breaks down Network_Termination_Case__c counts by NTC.Termination_Type__c ('Voluntary', 'Involuntary', 'Expiry') for the selected period.

Given the '90-Day Notice Compliance Rate' KPI tile,
When rendered,
Then it displays the percentage of NTC records where Termination_Type__c = 'Involuntary' AND Compliance_Notice_Status__c IN ('Acknowledged', 'Declined') AND Notice_Sent_Date__c <= Notice_Dispatch_Deadline__c, with a red threshold indicator when rate < 95%.

Given the 'Claims Suspended by Termination Event' table widget,
When rendered,
Then it lists each confirmed NTC with columns: Provider NPI, Network, Effective Date, Total Claims Suspended (count), and Total Payment Value Held (sum of Payment__c.Amount__c where Hold_Reason__c = 'Provider Termination Pending Review').

Given the 'Average Days to Confirm' metric tile,
When rendered,
Then it calculates the average of (NTC.Confirmed_Date__c - NTC.CreatedDate) in days for all NTCs where Status__c = 'Confirmed' in the selected period, displayed to one decimal place.

-Include a 'Provider Credentialing Lapse' sub-panel on this dashboard- (descoped to Sprint 11 per INS-0841 planning)

Given a Compliance Officer drills into a red-flagged compliance tile,
When they click the tile,
Then they are navigated to a filtered list view of Network_Termination_Case__c records matching the breach criteria.

## Solution

Build a CRM Analytics dashboard ('Provider_Termination_KPI_Dashboard') using SAQL queries against the Network_Termination_Case__c, Payment__c, and Claim__c datasets synced via the Analytics Data Manager. Configure nightly dataflow to refresh NTC, PCN, and Payment datasets. Use a global date range filter widget bound to NTC.CreatedDate for all lenses. Implement the compliance rate KPI lens with a SAQL conditional count formula. The claims-suspended table uses a joined dataset (NTC + Payment). Create a 'Network_Reporting_Viewer' Permission Set granting Analytics Asset access. Add a drill-through action on the compliance tile using a URL action pointing to the NTC filtered list view with pre-applied filter criteria passed as URL parameters.
