## Description

As a Provider Relations Specialist, I need the Provider 360 FlexCard on the Provider__c (Person Account) record page to display any active Network_Termination_Case__c with key exit timeline details so that I have immediate visibility into a provider's network exit status without navigating away from the provider record. Extends INS-0786.

## Acceptance Criteria

Given a Provider__c record with one or more Network_Termination_Case__c records where NTC.Status__c IN ('Initiated', 'In Review', 'Pending Termination')
When a Provider Relations Specialist opens the Provider 360 page
Then the 'Network Exit Status' section of the FlexCard renders with:
  - NTC.Status__c displayed as a color-coded badge (Initiated=Yellow, In Review=Orange, Pending Termination=Red)
  - NTC.Effective_Date__c formatted as MM/DD/YYYY
  - NTC.Termination_Reason__c
  - NTC.LOB__c
  - Days until termination calculated inline

Given a Provider__c with no active Network_Termination_Case__c
When the FlexCard loads
Then the 'Network Exit Status' section displays 'No Active Termination Cases' with a green indicator

Given a Provider__c with NTC.Status__c = 'Closed'
When the FlexCard loads
Then the closed termination case is NOT shown in the active section but is accessible via a 'View History' action link

-Given the FlexCard displays a 'Terminate Now' quick action button visible to all users-
-This requirement is superseded; the action is restricted to users with 'Provider_Termination_Ops' Permission Set Group only-

Given the user has 'Provider_Termination_Ops' Permission Set Group
When they click 'Initiate Exit Checklist' on the FlexCard
Then the Network Exit OmniScript (INS-0818) launches in a modal

## Solution

Extend the existing Provider 360 FlexCard (OmniStudio) by adding a new 'Network Exit Status' child card. Use a DataRaptor Extract to query Network_Termination_Case__c filtered by Provider__c = recordId and Status__c IN ('Initiated','In Review','Pending Termination'). Bind results to FlexCard state. Implement conditional rendering via Show/Hide conditions on badge components. Use a Formula element to compute days-until-termination from NTC.Effective_Date__c. Action button visibility controlled by a Custom Permission 'Can_Initiate_Exit_Checklist' assigned via Permission Set Group. Deploy FlexCard to Provider__c Lightning Record Page via App Builder.
