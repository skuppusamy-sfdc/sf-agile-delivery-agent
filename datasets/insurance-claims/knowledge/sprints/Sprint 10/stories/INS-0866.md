## Description

As a Compliance Officer, I need the system to automatically generate Provider_Change_Notification__c records and dispatch PCN letters to all affected policyholders via DocuSign when a Network_Termination_Case__c reaches Status__c = 'Approved for Termination', so that Acme Insurance meets the state-mandated 90-day advance notice requirement and avoids regulatory penalties. Extends INS-0864.

## Acceptance Criteria

Given a Network_Termination_Case__c.Status__c transitions to 'Approved for Termination',
When the record is saved,
Then a Provider_Change_Notification__c (PCN) record is created for each unique Policy__c linked to active Claim__c records where Claim.Rendering_Provider__c = the terminated provider's NPI, within 15 minutes.

Given a PCN record is created,
When the PCN batch dispatches,
Then PCN.Notification_Type__c = 'Provider Network Exit', PCN.Dispatch_Date__c = TODAY(), and PCN.Compliance_Deadline__c = TODAY() + 90.

Given a PCN record is ready for dispatch,
When the DocuSign integration is triggered via Apex (DocuSignPCNDispatcher.send()),
Then a DocuSign envelope is sent to Policy__c.Policyholder_Email__c with the Conga-generated PCN letter template (Template_ID: PCN_NETWORK_EXIT_v2) attached.

Given a DocuSign envelope is sent,
When the envelope status webhook returns 'Delivered' or 'Completed',
Then PCN.DocuSign_Status__c is updated accordingly and PCN.Acknowledgement_Timestamp__c is populated.

-Given dispatch fails, retry up to 3 times automatically- (deprecated: retry logic moved to dedicated retry queue in INS-0871)

Given PCN.Compliance_Deadline__c is within 10 days and PCN.DocuSign_Status__c != 'Completed',
When a daily scheduled Flow runs,
Then a High-priority Task is created assigned to the Compliance Officer role queue with subject 'URGENT: PCN Acknowledgement Overdue – [Policy Number]'.

## Solution

Record-Triggered Flow on Network_Termination_Case__c fires on Status__c change to 'Approved for Termination', invoking an Apex-invocable method (PCNGeneratorService.generatePCNs()) that queries affected Policy__c records via SOQL on Claim__c and bulk-inserts Provider_Change_Notification__c records. DocuSign dispatch handled via DocuSign for Salesforce managed package API, with Conga CLM template merge for letter generation. Webhook callback handled by a custom REST endpoint (@RestResource) updating PCN records. Compliance deadline monitoring via a Scheduled Flow (daily) querying overdue PCN records and creating Tasks via auto-launched Flow.
