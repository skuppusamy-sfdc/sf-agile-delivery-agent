## Description

As a Compliance Officer, I need the system to automatically generate and send a 90-day advance written termination notice to a provider via DocuSign when a Network_Termination_Case__c with Termination_Type__c = 'Involuntary' is set to Status__c = 'Notice Pending', so that Acme Insurance meets state-mandated notification requirements and maintains an auditable delivery record on the NTC record.

## Acceptance Criteria

Given a Network_Termination_Case__c where NTC.Termination_Type__c = 'Involuntary' and NTC.Status__c transitions to 'Notice Pending'
When the Record-Triggered Flow fires
Then a DocuSign envelope is generated using template 'Involuntary_Termination_Notice_v3' with merge fields:
  - Provider__c.Name
  - Provider__c.NPI__c
  - NTC.Effective_Termination_Date__c
  - NTC.Termination_Reason__c
  - PNA.LOB__c

Given the DocuSign envelope is sent
When the provider signs or the envelope status is 'Delivered'
Then NTC.Notice_Sent_Date__c = Today() and NTC.Notice_Delivery_Status__c = 'Delivered'

Given NTC.Effective_Termination_Date__c minus NTC.Notice_Sent_Date__c is less than 90 days
When the flow attempts to send the notice
Then the flow is halted, NTC.Compliance_Flag__c = TRUE, and an escalation task is created for the Compliance Officer

Given the DocuSign envelope bounces or delivery fails
When the callback webhook updates the envelope status to 'Failed'
Then NTC.Notice_Delivery_Status__c = 'Failed' and a high-priority Case is created linked to the NTC record

-Given voluntary terminations, also send 30-day notice via DocuSign- (removed per compliance review — voluntary notice handled by separate PCN flow INS-0836)

## Solution

Record-Triggered Flow on Network_Termination_Case__c triggers on Status__c change to 'Notice Pending' where Termination_Type__c = 'Involuntary'. Apex invocable class 'DocuSign_TerminationNotice_Handler' calls DocuSign eSignature REST API to create envelope from template ID stored in Custom Metadata (DocuSign_Template__mdt). 90-day validation logic implemented in a before-save Flow formula. DocuSign callback handled via a named @RestResource Apex class that updates NTC fields. DocuSign_Envelope_Id__c stored on NTC for audit trail. All operations logged to Compliance_Audit_Log__c.
