## Description

As a Provider Network Specialist, I need an automated cascade review workflow that identifies all Group_Service_Agreement__c records impacted when a provider is terminated from the network, so that Broker Relations Coordinators and employer group contacts can be proactively notified and GSA amendments can be initiated before the termination effective date.

## Acceptance Criteria

Given a Network_Termination_Case__c is created for a Provider__c,
When the NTC record is inserted,
Then a SOQL query identifies all Group_Service_Agreement__c records where GSA.Covered_Provider_NPI__c includes the terminated provider's NPI and GSA.Status__c = 'Active'.

Given impacted GSA records are identified,
When the count of impacted GSAs > 0,
Then NTC.Impacted_GSA_Count__c is populated and a child GSA_Impact_Review__c junction record is created for each GSA linking NTC.Id and GSA.Id with Review_Status__c = 'Pending'.

Given GSA_Impact_Review__c records are created,
When the workflow initiates,
Then an automated email notification is sent to GSA.Broker_Contact__c (lookup to Contact) and GSA.Employer_Group_Contact__c using the 'GSA_Provider_Termination_Notice' email template, with NTC.Effective_Termination_Date__c and Provider__c.Name merged into the body.

Given a Broker Relations Coordinator reviews a GSA_Impact_Review__c,
When they set Review_Status__c = 'Amendment Required',
Then a new Conga CLM contract amendment task is auto-created on the GSA record with Subject = 'GSA Amendment – Provider Network Exit' and Due_Date__c = NTC.Effective_Termination_Date__c - 30.

Given all GSA_Impact_Review__c records for a given NTC have Review_Status__c IN ('No Action Required', 'Amendment Complete'),
When the last review is closed,
Then NTC.GSA_Review_Complete__c = true is stamped automatically via a Roll-Up or Flow.

## Solution

Record-Triggered Flow on Network_Termination_Case__c (After Insert) calls an Apex-invocable method (GSAImpactAnalysisService.analyzeImpact()) that performs a SOQL query on Group_Service_Agreement__c using a semi-join on a related junction. GSA_Impact_Review__c is a custom junction object with Master-Detail relationships to both Network_Termination_Case__c and Group_Service_Agreement__c. Email notifications sent via Apex Messaging.sendEmail() using a custom EmailTemplate. Conga CLM amendment task creation via Conga REST API callout in an @future method. GSA review completion tracked via a Roll-Up Summary field or a Record-Triggered Flow using collection filtering on child GSA_Impact_Review__c records.
