## Description

As a Claims Operations Manager, I need a unified Policyholder 360 FlexCard on the Contact record so that I can view all active policies, open claims, recent payments, and coordination of benefits (COB) flags without navigating away from the policyholder record, reducing handle time during FNOL intake.

## Acceptance Criteria

Given a Contact record flagged as a Policyholder (Contact.RecordType = 'Policyholder'),
When the Claims Operations Manager opens the record,
Then the Policyholder 360 FlexCard renders within 3 seconds and displays all Policy__c records where Policy__c.Policyholder__c = Contact.Id AND Policy__c.Status__c IN ('Active', 'Grace Period').

Given an active Policy__c exists,
When the FlexCard policy panel is rendered,
Then each policy tile displays Policy__c.Policy_Number__c, Policy__c.LOB__c, Policy__c.Effective_Date__c, Policy__c.Expiration_Date__c, and Policy__c.Premium_Amount__c.

Given one or more Claim__c records exist for the policyholder,
When the Open Claims tab is selected on the FlexCard,
Then all Claim__c records where Claim__c.Status__c NOT IN ('Closed', 'Denied', 'Withdrawn') are displayed, showing Claim__c.Claim_Number__c, Claim__c.Date_of_Loss__c, Claim__c.LOB__c, and Claim__c.Total_Billed_Amount__c.

Given Payment__c records exist linked to the policyholder's claims,
When the Payment History tab is selected,
Then the last 12 months of Payment__c records are shown, sorted by Payment__c.Payment_Date__c descending, displaying Payment__c.Amount__c, Payment__c.Payment_Method__c, and Payment__c.Status__c.

Given a Contact record has COB_Flag__c = TRUE,
When the FlexCard header renders,
Then a prominent amber COB indicator badge is displayed with a tooltip showing Contact.COB_Notes__c.

-Given a TPA__c lookup exists on the policy, display TPA name inline on the policy tile- (superseded: TPA details moved to Policy Detail modal in INS-0081).

## Solution

Build an OmniStudio FlexCard ('Policyholder360Card') anchored to the Contact object with RecordType filter. Use DataRaptor Extract actions to query Policy__c, Claim__c, and Payment__c in parallel via Integration Procedures to minimize SOQL round-trips. Implement tabbed card layout using FlexCard child cards for Claims and Payments panels. COB badge driven by a conditional state on Contact.COB_Flag__c. Deploy via OmniStudio Deployment package; assign visibility via Permission Set Group 'Claims_Operations_PSG'. Cache Integration Procedure responses for 60 seconds using Platform Cache to meet the 3-second render SLA.
