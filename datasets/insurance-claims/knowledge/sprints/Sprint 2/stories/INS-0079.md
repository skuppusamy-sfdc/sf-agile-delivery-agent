## Description

As a Provider Relations Specialist, I need a Facility 360 page on the Facility__c (Business Account) record so that I can view all Group Service Agreements (GSAs), associated provider rosters, and rolling 90-day claims volume to support contract renewal and network adequacy reviews. Extends INS-0078 patterns for consistent 360 page architecture.

## Acceptance Criteria

Given an Account record with RecordType = 'Facility' (Business Account),
When the Provider Relations Specialist opens the record,
Then the Facility 360 Lightning page displays Account.Facility_Type__c, Account.NPI__c, Account.Accreditation_Status__c, and Account.Network_Tier__c.

Given one or more Group_Service_Agreement__c records exist,
When the GSA panel renders,
Then all GSAs where GSA__c.Facility__c = Account.Id AND GSA__c.Status__c IN ('Active', 'Pending Renewal') are listed with GSA__c.Agreement_Number__c, GSA__c.Effective_Date__c, and GSA__c.Annual_Value__c.

Given Claim__c records exist with Claim__c.Facility__c = Account.Id,
When the Claims Volume widget renders,
Then a summary card shows total count and sum of Claim__c.Total_Billed_Amount__c for claims with Claim__c.Date_of_Service__c in the rolling 90-day window, broken down by Claim__c.LOB__c.

Given the facility has more than 50 associated Provider__c records via the Facility_Provider__c junction object,
When the Provider Roster tab loads,
Then pagination is applied (25 records per page) and a search bar filters by Provider__c.Name or Account.NPI__c.

## Solution

Create a new Lightning Record Page for the Facility RecordType in App Builder. Build 'facilityGsaPanel' LWC using Apex controller with SOQL aggregate queries for claims volume (SUM, COUNT with GROUP BY LOB__c). Provider roster component uses server-side pagination via Apex with OFFSET/LIMIT and a @wire-connected search input. Reuse the PNA panel LWC architecture from INS-0078 as a base pattern. Claims volume KPI cards styled with SLDS utility classes. Apex class annotated with @AuraEnabled(cacheable=true) for read operations.
