## Description

As a Claims Operations Manager, I need the Policyholder 360 view to include an EOB history panel that lists all Explanation_of_Benefits__c records for the policyholder and allows staff to download the associated PDF from the linked DocuSign envelope so that policyholders' EOB records are accessible without leaving Salesforce. Extends INS-0108.

## Acceptance Criteria

Given a policyholder has one or more Explanation_of_Benefits__c records,
When the EOB History panel on the Policyholder 360 FlexCard loads,
Then all EOB__c records where EOB__c.Policyholder__c = current Contact Id must be listed sorted by EOB__c.EOB_Date__c DESC, showing EOB__c.Claim_Number__c, EOB__c.EOB_Date__c, EOB__c.Amount_Paid__c, and EOB__c.EOB_Status__c.

Given an EOB__c record has a non-null EOB__c.DocuSign_Envelope_Id__c,
When the user clicks 'Download PDF',
Then the system must call the DocuSign REST API via a named credential ('DocuSign_Prod') to retrieve the completed envelope document and stream it as a PDF download to the user's browser.

Given an EOB__c record has EOB__c.DocuSign_Envelope_Id__c = null,
When the Download PDF button is rendered,
Then the button must be disabled and display tooltip text 'PDF not yet available'.

-Given the EOB is older than 7 years, suppress it from the panel- (deprecated — retention filtering moved to a scheduled batch job per compliance review)

Given more than 20 EOB records exist,
When the panel renders,
Then pagination must be applied showing 10 records per page with Next/Previous controls.

## Solution

Extend the Policyholder360Card FlexCard (INS-0108) with a new 'EOB History' card block. DataRaptor Extract DR_GetEOBHistory queries Explanation_of_Benefits__c ordered by EOB_Date__c DESC. PDF download triggers an Apex @AuraEnabled method 'EOBDocuSignController.getEnvelopeDocument()' that performs a callout to DocuSign REST API /v2.1/accounts/{accountId}/envelopes/{envelopeId}/documents/combined using a Named Credential. Response is base64-decoded and returned to the FlexCard action as a downloadable blob. Pagination handled natively via FlexCard datatable pagination settings.
