## Description

As a Provider Network Specialist, I need a Facility 360 view on the Facility (Business Account) record so that I can review the Group Service Agreement (GSA), see all affiliated individual providers, and understand the facility's current claims volume at a glance.

Facilities are modeled as Business Accounts with Record Type 'Facility'. The view must surface Group_Service_Agreement__c (GSA) contract details, a roster of affiliated Provider Person Accounts via the Provider_Facility_Affiliation__c junction object, and a Claim__c volume summary segmented by Claim__c.Status__c.

## Acceptance Criteria

Given a Provider Network Specialist opens a Facility (Business Account) record,
When the Facility 360 FlexCard loads,
Then the 'GSA Summary' panel displays the active Group_Service_Agreement__c where GSA__c.Facility__c = current Account Id AND GSA__c.Status__c = 'Executed', showing GSA__c.Contract_Number__c, GSA__c.Effective_Date__c, GSA__c.Expiration_Date__c, GSA__c.Contracted_Rate_Basis__c.

Given the facility has affiliated providers,
When the 'Affiliated Providers' panel renders,
Then it lists all Provider Person Accounts linked via Provider_Facility_Affiliation__c.Facility__c = current Account Id AND Provider_Facility_Affiliation__c.Status__c = 'Active', displaying the provider's Name, Provider__c.NPI__c, Provider__c.Specialty__c, and Provider__c.Credentialing_Status__c.

Given the facility has associated claims,
When the 'Claims Volume' panel renders,
Then it displays a summary count of Claim__c records grouped by Claim__c.Status__c for the rolling 90-day window based on Claim__c.Date_of_Loss__c.

Given the GSA is expiring within 90 days,
When the GSA Summary panel renders,
Then an amber warning badge displays: 'GSA expires on {GSA__c.Expiration_Date__c} — renewal action required.'

## Solution

OmniStudio FlexCard 'Facility_360_Card' targeting Business Account records with RecordType.DeveloperName = 'Facility'. GSA panel uses DataRaptor Extract on GSA__c with status filter. Affiliated Providers panel uses DataRaptor Extract on Provider_Facility_Affiliation__c with a child relationship query to Provider__c fields. Claims Volume panel uses a DataRaptor Turbo Extract with a SOQL aggregate query (COUNT grouped by Status__c, filtered by Date_of_Loss__c >= LAST_N_DAYS:90). GSA expiry warning uses FlexCard conditional state evaluating a formula: TODAY() >= GSA__c.Expiration_Date__c - 90. FlexCard deployed on the Facility Business Account Lightning page layout.
