# INS-0095: Add Facility (Business Account) 360 view with GSA details, network participation, and linked individual providers

## Description

As a Provider Relations Specialist, I need a Facility 360 view on the Facility__c (Business Account) record page so that I can see the facility's active Group Service Agreements (GSAs), linked individual providers (via Provider__c affiliation), network tier, and fee schedule applicability to streamline contract management and claims adjudication support. Extends INS-0094.

## Acceptance Criteria

Given a Facility__c record where Account.RecordType = 'Facility', when the Facility 360 FlexCard loads, then it must display Facility__c.NPI__c, Facility__c.TIN__c, Facility__c.Facility_Type__c, Facility__c.Network_Status__c, and Facility__c.Primary_Address__c.

Given a facility with one or more GSAs, when the GSA panel renders, then Group_Service_Agreement__c records where GSA__c.Facility__c = Facility__c.Id and GSA__c.Status__c = 'Active' must display GSA__c.Agreement_Number__c, GSA__c.Effective_Date__c, GSA__c.Expiration_Date__c, and GSA__c.Contracted_Services__c.

Given a facility with affiliated individual providers, when the Affiliated Providers panel renders, then Provider__c records linked via AccountContactRelation where AccountContactRelation.AccountId = Facility__c.Id must be listed with Provider__c.NPI__c, Provider__c.Primary_Specialty__c, and Provider__c.Network_Status__c.

-Given a facility with multiple LOBs under one GSA, display each LOB as a separate row with its own fee schedule.- (Requirement moved to Fee Schedule Engine story INS-0099)

Given a facility where GSA__c.Expiration_Date__c < TODAY + 60 days, then a red 'Contract Expiring' badge must render on the GSA row.

## Solution

Extend the OmniStudio FlexCard pattern established in INS-0094. Create 'Facility360Card' FlexCard with DataRaptor Turbo Extracts: DR_GetActiveGSAs (query Group_Service_Agreement__c filtered by Facility__c), DR_GetAffiliatedProviders (query AccountContactRelation + Provider__c fields). Use FlexCard conditional state for 'Contract Expiring' badge based on date offset formula. Reuse Permission Set Group 'Provider_Network_Read' for FLS. Add Facility360Card to Facility (Business Account) Lightning Record Page via App Builder. Coordinate with INS-0094 assignee on shared DataRaptor naming conventions.
