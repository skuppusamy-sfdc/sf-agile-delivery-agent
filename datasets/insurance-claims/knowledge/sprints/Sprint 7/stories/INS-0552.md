## Description

As a Claims Operations Manager, I need a dedicated Salesforce Lightning Dashboard displaying real-time claims payment disbursement KPIs and ERA transmission status so that I can monitor operational health, identify bottlenecks in the payment cycle, and report on SLA compliance to leadership on a daily basis.

## Acceptance Criteria

Given the Claims Operations Manager navigates to the 'Claims Payment Operations' Lightning Dashboard
When the dashboard loads
Then the following report components must be visible: (1) Total Payment__c.Amount__c disbursed in the current calendar month grouped by LOB, (2) Count of Payment__c records by Payment__c.Status__c ('Pending Disbursement', 'Disbursed', 'Failed') as a donut chart, (3) Average days from Claim__c.Adjudicated_Date__c to Payment__c.Disbursed_Date__c as a metric component, (4) ERA_Batch__c transmission success vs failure rate for the rolling 30 days as a bar chart, (5) Count of EOB__c records by EOB__c.Status__c including 'Generation Failed' as a highlight panel

Given the dashboard is filtered by LOB
When the Claims Operations Manager selects a specific LOB value from the dashboard filter
Then all report components must dynamically filter to show data for the selected LOB only, using a Dashboard Filter mapped to Payment__c.LOB__c and Claim__c.LOB__c

Given a Payment__c.Status__c = 'Failed' count exceeds the threshold defined in Custom Metadata (Dashboard_Alert_Config__mdt.Failed_Payment_Threshold__c)
When the dashboard renders the payment status donut chart
Then the 'Failed' segment must display in red (#FF0000) using conditional highlighting in the report

-Given the dashboard must support export to Excel via a custom LWC button- (descoped: standard Salesforce report export deemed sufficient per INS-0538 decision log)

Given the dashboard is scheduled for daily email delivery
When the Salesforce scheduled report subscription runs at 07:00 local time each business day
Then the dashboard snapshot email must be delivered to the Claims_Operations_Managers public group with subject line 'Daily Claims Payment Operations Report - [Date]'

## Solution

Create five custom Report Types in Salesforce: Payment Summary (Payment__c with Claim__c), ERA Transmission (ERA_Batch__c), and EOB Status (EOB__c with Claim__c). Build corresponding Salesforce Reports with appropriate groupings, summary formulas, and conditional highlighting rules. Assemble Lightning Dashboard 'Claims Payment Operations' with five components mapped to reports. Dashboard Filter created on LOB field. Scheduled dashboard subscription configured for Claims_Operations_Managers public group via Salesforce native Subscribe feature. Dashboard_Alert_Config__mdt queried by a Dashboard Refresh Flow to dynamically update conditional highlight thresholds if needed in future iterations.
