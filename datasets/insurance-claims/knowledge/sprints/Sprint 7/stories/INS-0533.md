## Description

As a Provider Relations Specialist, I need a consolidated Payment and EOB history panel on the Provider 360 FlexCard so that I can quickly review a provider's recent disbursements, outstanding payments, and remittance history during provider calls or dispute resolution without navigating multiple Salesforce objects.

## Acceptance Criteria

Given a Provider__c (Person Account) record page with the Provider 360 FlexCard
When the 'Payments & EOB' tab is selected
Then the FlexCard displays the 10 most recent Payment__c records related to the Provider__c, sorted by Payment__c.Payment_Date__c DESC, showing: Payment__c.Payment_Amount__c, Payment__c.Status__c, Payment__c.Payment_Method__c, and linked Claim__c.Claim_Number__c

Given a Payment__c with Payment__c.Status__c = 'Disbursed'
When displayed in the FlexCard panel
Then a 'View EOB' action button appears that opens the associated Explanation_of_Benefits__c ContentDocument PDF in a modal

Given a Provider__c with no Payment__c records in the last 90 days
When the Payments & EOB tab loads
Then an empty state message displays: 'No payments processed in the last 90 days'

Given a Provider__c with Payment__c.Status__c = 'Pending' or 'Failed'
When these records appear in the FlexCard
Then they are visually badged with color indicators: yellow for 'Pending', red for 'Failed' using OmniStudio conditional formatting

Given the FlexCard is loaded by a user without the 'Provider_Relations_Specialist' Permission Set Group
When access is attempted
Then the Payments & EOB tab is hidden and no Payment__c data is exposed

## Solution

OmniStudio FlexCard ('Provider360_PaymentEOB_Card') with a DataRaptor Extract as the data source, querying Payment__c WHERE Provider__c = :recordId ORDER BY Payment_Date__c DESC LIMIT 10. Child DataRaptor for EOB ContentDocument retrieval via ContentDocumentLink. Conditional formatting rules in FlexCard designer for status badge colors. 'View EOB' action uses a FlexCard Action of type 'Navigate' opening a modal LWC (eobPdfViewer) that fetches ContentVersion body via Apex controller. Visibility controlled via OmniStudio FlexCard Visibility Rules tied to the user's Permission Set Group assignment.
