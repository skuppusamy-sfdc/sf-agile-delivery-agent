## Description

As a Claims Operations Manager, I need an Explanation_of_Benefits__c (EOB) record and corresponding PDF document to be generated automatically when a Payment__c.Status__c transitions to 'Issued' so that members receive timely, compliant EOB communications detailing their claim adjudication outcome, approved amounts, denied line items, and member responsibility. The EOB must comply with ACA and state-specific disclosure mandates.

## Acceptance Criteria

Given a Payment__c.Status__c transitions from 'Pending' to 'Issued',
When the Record-Triggered Flow fires,
Then an Explanation_of_Benefits__c record is created with EOB__c.Claim__c = Payment__c.Claim__c, EOB__c.Payment__c = Payment__c.Id, EOB__c.Member_Name__c populated from Policy__c.Policyholder__r.Name, EOB__c.Date_Issued__c = TODAY(), and EOB__c.Status__c = 'Generated'.

Given the EOB__c record is created,
When Conga CLM template 'EOB_Standard_v3' is invoked,
Then a PDF is rendered containing: Claim__c.Claim_Number__c, all associated Claim_Line__c records with Claim_Line__c.CPT_Code__c, Claim_Line__c.Billed_Amount__c, Claim_Line__c.Allowed_Amount__c, Claim_Line__c.Denied_Amount__c, Claim_Line__c.Denial_Reason_Code__c, and Claim_Line__c.Member_Responsibility__c.

Given EOB__c.Delivery_Preference__c = 'Email',
When the PDF is generated,
Then it is attached to the EOB__c record as a ContentVersion and emailed to Policy__c.Policyholder__r.Email within 24 hours of Payment__c issuance.

Given EOB__c.Delivery_Preference__c = 'Mail',
When the PDF is generated,
Then EOB__c.Status__c = 'Pending Mail Fulfillment' and a Print_Fulfillment_Request__c record is created for the third-party mail vendor integration queue.

-Given EOB delivery via fax, system triggers fax API call- (Deprecated: fax delivery sunset per compliance directive CD-2024-11)

Given any Claim_Line__c with Claim_Line__c.Status__c = 'Denied',
When the EOB PDF is rendered,
Then the denial section must include the ICD-10 diagnosis code, denial reason in plain language, and the member's appeal rights statement per state regulatory requirements.

Given EOB__c is successfully delivered,
When delivery confirmation is received,
Then EOB__c.Status__c = 'Delivered' and EOB__c.Delivered_Date__c = confirmed delivery timestamp.

## Solution

Record-Triggered Flow on Payment__c (After Save, Status__c = 'Issued') creates EOB__c record. Invoke Conga CLM via REST callout from an Apex-invocable action (CongaEOBGenerator.generateEOB()) using template ID stored in Custom Metadata Type (EOB_Template_Config__mdt). PDF stored as ContentVersion linked to EOB__c via ContentDocumentLink. Email delivery via OrgWideEmailAddress-configured Messaging.SingleEmailMessage in Apex. Print fulfillment via outbound Platform Event EOB_Print_Request__e consumed by middleware. Claim_Line__c data fetched via DataRaptor Extract (DR_Get_ClaimLines_For_EOB) for rendering. All operations wrapped in try-catch with error logging to EOB_Generation_Error__c.
