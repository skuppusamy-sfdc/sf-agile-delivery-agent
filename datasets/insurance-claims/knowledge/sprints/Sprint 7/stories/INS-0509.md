## Description

As a Claims Operations Manager, I need a real-time Salesforce dashboard surfacing key payment disbursement KPIs — including total payments pending, payments on hold, EFT vs. check split, ERA transmission failures, and average days-to-payment per LOB — so that I can proactively identify bottlenecks and ensure SLA compliance for provider and policyholder disbursements.

## Acceptance Criteria

Given the Claims Operations Manager navigates to the 'Claims Payment Operations' Lightning App,
When the dashboard loads,
Then the following report-backed components are displayed: (1) Payment__c count by Status__c (donut chart), (2) Total Payment__c.Amount__c by Payment_Method__c (bar chart), (3) ERA Transmission Failures count (metric tile, filtered to current week), (4) Average Payment__c.Days_to_Payment__c by Claim__c.Line_of_Business__c (table), (5) Payments On Hold by Hold_Reason__c (list).

Given Payment__c.Status__c = 'On Hold',
When the 'Payments On Hold' dashboard component is viewed,
Then clicking any row navigates the user directly to the Payment__c record with the related Claim__c and Provider__c visible in the highlights panel.

Given the dashboard is open,
When data changes in the underlying Payment__c records,
Then the dashboard refreshes automatically every 5 minutes (using Lightning Dashboard auto-refresh setting).

-Given a dedicated Visualforce page for legacy payment reporting — DEPRECATED: replaced by Lightning dashboard per platform modernization mandate Sprint 6-

## Solution

Build five custom Salesforce Reports (Payment_Status_Summary, Payment_Method_Split, ERA_Failure_Weekly, Avg_Days_to_Payment_by_LOB, OnHold_by_Reason) using Payment__c as the primary object with cross-filters and bucket fields. Assemble into a Lightning Dashboard ('Claims Payment Operations Dashboard') with auto-refresh enabled. Dashboard scoped to 'Claims Operations Manager' Permission Set Group via folder sharing. Payment__c.Days_to_Payment__c is a formula field: Disbursement_Date__c - Claim__c.Adjudication_Date__c. Custom Lightning App 'Claims Payment Operations' configured with the dashboard as the default home tab.
