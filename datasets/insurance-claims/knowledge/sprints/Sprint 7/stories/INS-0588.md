## Description

As a Claims Operations Manager, I need a real-time Payment Disbursement Dashboard on the Salesforce home page so that I can monitor payment queue status, identify bottlenecks, track daily disbursement totals by LOB, and proactively manage SLA compliance for payment processing.

## Acceptance Criteria

Given the Claims Operations Manager accesses the Salesforce Home Page
When the 'Payment Disbursement Dashboard' tab is selected
Then the dashboard displays the following report components: (1) Payment Queue by Status (Pending/Approved/Transmitted/Failed) as a donut chart, (2) Daily Disbursement Total by LOB__c as a bar chart, (3) Payments Pending > 48 Hours list view, (4) ERA Transmission Failure count as a KPI metric tile

Given the dashboard is rendered
When the user selects a LOB filter (e.g., 'Commercial', 'Medicare Advantage', 'Medicaid')
Then all dashboard components refresh to reflect only Payment__c records where Payment__c.LOB__c matches the selected value

Given Payment__c records where Status__c = 'Pending' and CreatedDate < TODAY() - 2
When the 'Payments Pending > 48 Hours' list view component renders
Then it displays Claim__c.Claim_Number__c, Provider__c.Name, Payment__c.Payment_Amount__c, and Payment__c.CreatedDate sorted by CreatedDate ascending

-Given a separate 'TPA Payment Dashboard' is required for TPA-administered claims- (descoped to Phase 2 per stakeholder review 2024-11-20)

Given the dashboard data
When the page loads
Then all report data reflects records as of the last 15-minute refresh cycle (CRM Analytics or standard report with auto-refresh enabled)

## Solution

Build a Salesforce CRM Analytics (Tableau CQ) dashboard using SAQL queries on Payment__c and ERA_File__c datasets. Alternatively, implement as a Lightning Dashboard with 4 report components if CRM Analytics license is unavailable (to be confirmed with platform team). LOB filter implemented as a Dashboard Filter component bound to Payment__c.LOB__c. The 'Pending > 48 Hours' component uses a custom report type joining Payment__c and Claim__c. Dashboard is added to the Claims Operations Manager App via App Manager. Sharing via Folder shared to 'Claims Operations' Public Group.
