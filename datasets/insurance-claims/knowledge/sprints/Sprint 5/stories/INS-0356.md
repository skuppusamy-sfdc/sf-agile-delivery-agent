## Description

As a Compliance Officer, I need a monthly automated report that calculates the ratio of in-network providers to total providers by LOB and geographic region (state), so that Acme Insurance can demonstrate network adequacy compliance to state regulators and MCO partners. The report must be generated on the first business day of each month and delivered to the Compliance team distribution list.

## Acceptance Criteria

Given the first business day of each calendar month
When the scheduled report job executes
Then a Report (type: Summary) must aggregate Provider__c records grouped by Provider__c.State__c and Provider__c.Primary_LOB__c, with a calculated column: In_Network_Count (count of providers with at least one active PNA) / Total_Provider_Count * 100 = Network_Adequacy_Ratio__c.

Given the report runs
When any state/LOB combination has a Network_Adequacy_Ratio below 70%
Then a high-priority Case must be auto-created with Subject = 'Network Adequacy Below Threshold — [State] — [LOB]', RecordType = 'Compliance_Issue', and assigned to the Compliance Officer queue.

Given the report is generated
When it is complete
Then the report output must be exported as a CSV file attached to a custom Report_Archive__c record and an email with the CSV attachment dispatched to the Compliance_Distribution_List@acmeinsurance.com alias.

Given the report data is displayed in a Salesforce Dashboard
When a Compliance Officer views the 'Network Adequacy Dashboard'
Then gauge chart components must display current-month ratios per LOB with threshold markers at 70% (warning) and 85% (target).

## Solution

Create a Salesforce Report (Report Type: Provider with Provider Network Agreements) with Summary groupings on Provider__c.State__c and Provider__c.Primary_LOB__c. A Scheduled Apex class (NetworkAdequacyReportScheduler) runs on the first business day (calculated via BusinessHours API) to: (1) execute the report via Analytics REST API, (2) parse results, (3) create Case records for below-threshold combinations via DML, (4) generate CSV via String serialization and attach to Report_Archive__c via ContentVersion + ContentDocumentLink, (5) send email via Messaging.SingleEmailMessage with CSV as blob attachment. Dashboard built in native Salesforce with gauge charts referencing the Summary Report. Permission Set (PS_Compliance_Reporting) grants access to dashboard and report folder.
