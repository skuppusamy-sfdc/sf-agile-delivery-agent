## Description

As a Claims Operations Manager, I need a real-time Salesforce dashboard surfacing claim volume, adjudication cycle times, denial rates, and pending FNOL queues by LOB so that I can proactively manage team workload and identify processing bottlenecks before they impact SLA compliance.

This dashboard will serve as the primary operational view for sprint 16 reporting deliverables and extends foundational KPI work established in INS-1393.

## Acceptance Criteria

Given the Claims Operations Manager navigates to the 'Claims Ops Dashboard' tab,
When the page loads,
Then KPI tiles display the following metrics refreshed within the last 15 minutes:
  - Total open Claim__c records where Claim__c.Status__c IN ('Submitted','In Review','Pending Info')
  - Average Claim__c.Adjudication_Cycle_Days__c grouped by Claim__c.Line_of_Business__c
  - Denial rate = COUNT(Claim__c where Status__c = 'Denied') / COUNT(all Claim__c) for rolling 30 days
  - FNOL queue count where Claim__c.FNOL_Received__c = true AND Claim__c.Status__c = 'New'

Given the dashboard is rendered,
When the manager selects a LOB filter (e.g., 'Commercial', 'Medicare Advantage', 'Medicaid'),
Then all tiles and charts filter dynamically without page reload and Claim__c.Line_of_Business__c drives the filter predicate.

Given the adjudication trend chart is visible,
When the manager hovers over a data point,
Then a tooltip displays the exact date, Claim__c count, and average Claim__c.Adjudication_Cycle_Days__c for that period.

Given the dashboard is accessed by a user without the 'Claims_Ops_Manager' Permission Set Group,
When they attempt to navigate to the tab,
Then they receive a 'You do not have access to this dashboard' message and the underlying report data is not exposed.

Given the denial rate tile,
When the denial rate exceeds 15% for any LOB,
Then the tile background renders in red (#D9534F) and a warning icon is displayed.

## Solution

Build using CRM Analytics (Tableau CRM) with a dedicated 'Claims_Ops_Dashboard' app. Create SAQL-based datasets joining Claim__c, Claim_Line__c, and Payment__c objects via DataSync. KPI tiles implemented as Compare Table widgets. Adjudication trend chart uses a Time Series widget grouped by Claim__c.CloseDate. LOB filter implemented as a Global Filter widget bound to Claim__c.Line_of_Business__c. Row-level security enforced via a Security Predicate on the dataset referencing the user's Permission Set Group membership. Dashboard tab visibility controlled via a custom App in App Manager with Profile/Permission Set access. Conditional tile coloring achieved via SAQL CASE expressions and widget color threshold configuration. Schedule dataset refresh every 15 minutes via a Scheduled Dataflow.
