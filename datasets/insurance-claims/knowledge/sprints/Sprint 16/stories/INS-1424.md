## Description

As a Provider Network Specialist, I need a Salesforce report and dashboard component that shows all Provider_Network_Agreement__c (PNA) records expiring within 90 days, grouped by negotiation status and assigned specialist, so that I can prioritize contract renewal outreach and avoid network gap events that could trigger compliance violations.

This story covers the report layer; the associated FlexCard surface for Provider 360 is tracked in INS-1426.

## Acceptance Criteria

Given the Provider Network Specialist runs the 'PNA Expiration Pipeline' report,
When the report executes,
Then it returns all Provider_Network_Agreement__c records where Provider_Network_Agreement__c.Contract_End_Date__c <= TODAY() + 90 AND Provider_Network_Agreement__c.Status__c != 'Terminated' AND Provider_Network_Agreement__c.Status__c != 'Expired'.

Given the report results,
When grouped by Provider_Network_Agreement__c.Negotiation_Status__c,
Then the following grouping values are present: 'Not Started', 'In Negotiation', 'Pending Signature', 'Executed — Pending Activation'.

Given the report is rendered,
When the specialist clicks a PNA record row,
Then they are navigated directly to the Provider_Network_Agreement__c record detail page.

Given the report is added to the Provider Network Dashboard,
When displayed as a dashboard component,
Then it renders as a stacked bar chart with X-axis = expiration month bucket (30/60/90 days) and Y-axis = count of PNA records, segmented by Negotiation_Status__c.

Given a PNA record where Provider_Network_Agreement__c.Contract_End_Date__c <= TODAY() + 30,
When displayed in the report,
Then the row is highlighted with a 'Critical' label and the assigned Provider_Network_Specialist__c field is prominently displayed for escalation routing.

-Given the report includes Fee_Schedule__c linked rate details inline- (deprecated — fee schedule detail moved to separate FeeScheduleEngine report per architecture review)

## Solution

Create a custom Salesforce Report Type with Provider_Network_Agreement__c as the primary object, joined to Provider__c (Person Account) and Facility__c (Business Account) via lookup relationships. Build the report using the standard Report Builder with custom summary formulas for 30/60/90-day bucket columns using IF(AND()) logic on Contract_End_Date__c. Add the report as a Dashboard component (bar chart) on the 'Provider Network Operations' Dashboard. Row-level access controlled via Sharing Rules on Provider_Network_Agreement__c based on OwnerId matching the specialist's user record. Conditional highlighting configured via report Conditional Formatting rules. Schedule report refresh daily at 06:00 AM local time via scheduled report subscription.
