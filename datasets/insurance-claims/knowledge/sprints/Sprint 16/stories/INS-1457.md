## Description

As a Claims Operations Manager, I need Agentforce-powered next-best-action (NBA) recommendation cards displayed on open Claim__c records so that claims adjusters receive AI-guided suggestions — such as requesting additional documentation, escalating to SIU, or initiating COB verification — reducing average handle time and improving first-pass adjudication rates.

This story covers the configuration of Einstein Next Best Action strategies, the LWC recommendation card component, and the Agentforce topic/action definitions for claims triage.

## Acceptance Criteria

Given an adjuster opens a Claim__c record with Status__c = 'In Review',
When the 'AI Recommendations' LWC panel renders on the record page,
Then Einstein Next Best Action surfaces 1–3 ranked recommendation cards based on the active NBA Strategy 'Claims_Triage_Strategy'.

Given Claim__c.Total_Billed_Amount__c > 50000 AND Claim__c.SIU_Referral__c = FALSE,
When the NBA strategy evaluates,
Then a recommendation card titled 'Consider SIU Referral' is surfaced with a confidence score >= 0.75, and an action button 'Refer to SIU' that sets Claim__c.SIU_Referral__c = TRUE and creates a linked Case record with RecordType = 'SIU Investigation'.

Given Claim__c.COB_Indicator__c = TRUE AND Claim__c.COB_Verification_Status__c = 'Pending',
When the NBA strategy evaluates,
Then a recommendation card titled 'Initiate COB Verification' is surfaced, with an action button launching the 'COB_Verification' OmniScript pre-populated with Claim__c.Policy__c and Claim__c.Claimant__c.

Given Claim__c.Missing_Documents__c > 0,
When the NBA strategy evaluates,
Then a recommendation card titled 'Request Missing Documentation' is surfaced listing Claim__c.Missing_Document_List__c values, with a 'Send DocuSign Request' action button.

Given an adjuster dismisses a recommendation card,
When the dismiss action fires,
Then a NBA_Feedback__c record is created with fields: Claim__c (lookup), Recommendation_Title__c, Action_Taken__c = 'Dismissed', Dismissed_By__c, Dismissed_DateTime__c — for model retraining telemetry.

Given the Agentforce topic 'Claims Triage Assistant' is invoked via the Agentforce sidebar,
When the adjuster types 'What should I do with this claim?',
Then the agent responds with the top NBA recommendation in natural language and offers to execute the suggested action.

-Given NBA recommendations also surface on Claim_Line__c records, Then line-level recommendations appear in a sub-panel- (descoped — line-level AI deferred to Phase 3)

## Solution

Configure Einstein Next Best Action with a Flow-based Recommendation Strategy 'Claims_Triage_Strategy' using branching logic on Claim__c fields (Total_Billed_Amount__c, COB_Indicator__c, Missing_Documents__c, SIU_Referral__c). Recommendations defined as Recommendation__c records with associated Action Flows. LWC 'claimsAiRecommendationPanel' consumes the @wire ConnectApi.RecommendationStrategies adapter and renders cards using SLDS Card components. Agentforce Agent topic 'Claims Triage Assistant' configured in Agent Builder with a custom Apex Action 'GetClaimsNBARecommendations' that wraps the NBA strategy invocation. NBA_Feedback__c object created with a Record-Triggered Flow on dismiss button click. DocuSign action uses the existing DocuSign for Salesforce managed package trigger. Permission Set Group 'Agentforce_Claims_User' gates access.
