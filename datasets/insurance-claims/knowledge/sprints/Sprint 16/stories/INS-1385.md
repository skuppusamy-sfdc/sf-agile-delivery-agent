## Description

As a Provider Network Specialist, I need a scheduled report and dashboard component showing all Provider_Network_Agreement__c (PNA) records expiring within 30, 60, and 90-day thresholds so that I can prioritize contract renewal outreach and prevent provider network gaps that would disrupt claims adjudication. Extends INS-1354 (PNA lifecycle management flow).

## Acceptance Criteria

Given a Provider_Network_Agreement__c record where PNA__c.Contract_End_Date__c is within 90 calendar days of TODAY,
When the 'PNA Expiration Watch' report runs (scheduled daily at 06:00 AM CT),
Then the report returns the record and populates the following columns:
  - Provider__c.Name (lookup to Person Account)
  - Provider__c.NPI__c
  - Provider__c.TIN__c
  - PNA__c.Contract_End_Date__c
  - PNA__c.LOB__c
  - PNA__c.Network_Tier__c
  - PNA__c.Assigned_Specialist__c
  - Days_Until_Expiration__c (formula field)

Given the report is rendered on the Provider Network Dashboard,
When grouped by expiration bucket,
Then records are bucketed as '0-30 Days' (highlighted red), '31-60 Days' (highlighted amber), '61-90 Days' (highlighted yellow) using report bucketing on Days_Until_Expiration__c.

Given a PNA record where PNA__c.Status__c = 'Renewed' or 'Terminated',
When the report executes,
Then that record is excluded from all expiration buckets.

Given the report runs,
When there are PNA records linked to a Facility__c (Business Account) rather than a Provider__c (Person Account),
Then Facility__c.Name and Facility__c.NPI__c are displayed in the respective provider columns.

Given the scheduled report completes,
When the '0-30 Days' bucket contains 1 or more records,
Then an automated email alert is dispatched to the record's PNA__c.Assigned_Specialist__c and their manager via Salesforce Email Alert action on the scheduled report subscription.

## Solution

Create a custom Salesforce Report Type joining Provider_Network_Agreement__c with Provider__c (Person Account) and Facility__c (Business Account) via polymorphic lookup. Build the 'PNA Expiration Watch' report using this report type with bucket fields on Days_Until_Expiration__c (formula: PNA__c.Contract_End_Date__c - TODAY()). Add report to 'Provider Network Ops' dashboard as a donut chart component. Configure a Report Subscription with a conditional threshold trigger (bucket '0-30 Days' count > 0) to fire an Email Alert. The Days_Until_Expiration__c formula field is deployed on Provider_Network_Agreement__c via metadata. Sharing on PNA records governed by Criteria-Based Sharing Rules aligned to PNA__c.Assigned_Specialist__c ownership.
