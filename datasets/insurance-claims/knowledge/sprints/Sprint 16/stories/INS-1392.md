## Description

As a Claims Operations Manager, I need a real-time Salesforce dashboard that surfaces key performance indicators including average adjudication cycle time, denial rate by LOB, pending FNOL volume, and total payment disbursed (MTD) so that I can proactively identify bottlenecks and drive operational efficiency across the claims processing team.

## Acceptance Criteria

Given the Claims Operations Manager navigates to the 'Claims Ops Command Center' dashboard,
When the page loads,
Then the following KPI tiles must be visible and populated with live data:
  - 'Avg Adjudication Cycle Time (Days)' sourced from Claim__c.Adjudication_Cycle_Days__c
  - 'Denial Rate %' calculated as (COUNT of Claim__c where Status__c = 'Denied') / (COUNT of all Claim__c) * 100
  - 'Open FNOL Volume' sourced from COUNT of Claim__c where Status__c = 'FNOL Received'
  - 'MTD Payments Disbursed ($)' sourced from SUM of Payment__c.Amount__c where Payment__c.Payment_Date__c = current month

Given the dashboard is filtered by LOB,
When the Claims Operations Manager selects a specific LOB value (e.g., 'Commercial', 'Medicare Advantage', 'Workers Comp'),
Then all KPI tiles and report charts must refresh to reflect data scoped to that LOB via Claim__c.Line_of_Business__c

Given the 'Denial Rate by LOB' bar chart is rendered,
When the user hovers over a bar,
Then a tooltip must display the absolute denial count and denial rate percentage for that LOB

Given a KPI tile value breaches a configured threshold (e.g., Avg Cycle Time > 14 days),
When the tile renders,
Then the tile background must display in amber (#FF8C00) or red (#D32F2F) per threshold configuration stored in Custom Metadata Type Reporting_Threshold__mdt

-Given the dashboard includes a 'SIU Referral Rate' tile sourced from Claim__c.SIU_Referred__c = true- (deprecated: moved to INS-1401 SIU Analytics epic)

Given the user has the 'Claims_Ops_Manager' Permission Set Group assigned,
When they access the dashboard,
Then they must see all tiles; users without this PSG must see a restricted view with MTD Payments tile hidden

## Solution

Create a Salesforce CRM Analytics (Tableau CRM) dashboard 'Claims Ops Command Center' with SAQL-based KPI widgets pulling from Claim__c and Payment__c datasets synced via dataflow. Implement Dynamic Dashboard with running-user filter. Threshold coloring handled via conditional formatting rules in Analytics Studio. Permission visibility enforced via Analytics Asset Sharing with Permission Set Group 'Claims_Ops_Manager'. LOB filter implemented as a Global Filter widget bound to Claim__c.Line_of_Business__c picklist. Custom Metadata Type Reporting_Threshold__mdt stores configurable threshold values consumed by SAQL case expressions.
