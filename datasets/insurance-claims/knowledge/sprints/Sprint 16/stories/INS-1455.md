## Description

**Bug Report — Claims Payment Summary Dashboard Widget**

**Environment:** Acme Insurance Production Sandbox (Sprint 16 deploy)

**Steps to Reproduce:**
1. Navigate to the 'Claims Payment Summary' CRM Analytics dashboard.
2. Apply filter: Claim__c.COB_Indicator__c = TRUE.
3. Observe the 'Total Adjudicated Amount' KPI tile value.
4. Independently run a Salesforce report summing Claim_Line__c.Adjudicated_Amount__c for the same filtered Claim__c set.
5. Compare values.

**Expected Behavior:**
The 'Total Adjudicated Amount' KPI tile should equal the SUM of Claim_Line__c.Adjudicated_Amount__c for all Claim__c records where COB_Indicator__c = TRUE, reflecting the net-of-COB adjudicated value.

**Actual Behavior:**
The KPI tile returns a value approximately 2x the correct amount. Investigation suggests the SAQL query is joining Claim_Line__c to both the primary Claim__c dataset and the COB_Claim__c lookup dataset, causing duplicate row counts for COB claims.

**Impact:** Claims Operations Managers are seeing inflated disbursement figures, leading to incorrect financial reporting. Severity: High — affects end-of-month close reporting.

**Linked records:** Reproducible with Claim__c records: CLM-0048821, CLM-0049103.

## Acceptance Criteria

Given the fix is applied,
When non-COB claims are filtered (Claim__c.COB_Indicator__c = FALSE),
Then the KPI tile value is unchanged from pre-fix baseline (regression check).

## Solution

Root cause: the CRM Analytics dataflow join node between Claim__c and Claim_Line__c datasets uses an 'outer' join type without deduplication, and the COB_Claim__c lookup relationship creates a second join path inflating row counts. Fix: modify the dataflow Augment node to use 'lookup' join type (not 'left-outer') and add a SAQL COALESCE deduplication step on Claim_Line__c.Id before aggregation. Additionally, add a SAQL WHERE clause guard: `q = filter q by 'COB_Secondary_Processed__c' == "false"` to exclude already-netted secondary lines. Unit test with CLM-0048821 and CLM-0049103 before promoting to UAT.
