## Description

As a Claims Operations Manager, I need an analytics report that tracks Explanation_of_Benefits__c delivery outcomes segmented by delivery channel (Mail, Email, Portal) and failure reason so that I can identify systemic delivery failures, ensure regulatory EOB delivery compliance, and prioritize remediation efforts.

## Acceptance Criteria

Given the 'EOB Delivery Analytics' report is run,
When results are returned,
Then the report must display: Explanation_of_Benefits__c.Name, EOB_Delivery_Channel__c, EOB_Delivery_Status__c, EOB_Delivery_Failure_Reason__c, Claim__r.Claim_Number__c, Policy__r.Policy_Number__c, EOB_Generated_Date__c, EOB_Delivered_Date__c

Given the report is grouped by EOB_Delivery_Channel__c,
When summarized,
Then each channel group must show COUNT of total EOBs, COUNT of EOB_Delivery_Status__c = 'Delivered', COUNT of EOB_Delivery_Status__c = 'Failed', and a calculated Delivery Success Rate % column

Given EOB_Delivery_Status__c = 'Failed',
When the report is filtered to failures only,
Then EOB_Delivery_Failure_Reason__c must be populated and visible; records where this field is null and status is 'Failed' must be flagged with a conditional highlight (red row)

-Given the report includes ERA reconciliation status column sourced from Payment__c.ERA_Status__c- (deprecated: ERA reconciliation reporting moved to Payment Analytics dashboard INS-1397)

Given a Compliance Officer exports the report,
When exported to CSV,
Then PII fields (Policy__r.Member_Name__c) must be masked per the 'Compliance_Officer' Permission Set field-level security configuration

## Solution

Build a Salesforce joined report across Explanation_of_Benefits__c (primary) and Claim__c / Policy__c via lookup relationships. Add formula column for Delivery Success Rate % using summary formula: (Delivered_Count / Total_Count) * 100. Conditional highlighting rule configured for null EOB_Delivery_Failure_Reason__c on failed records. FLS on Policy__r.Member_Name__c restricted for Compliance Officer profile via Permission Set. Report subscribed to Claims Ops Manager role for weekly email delivery. Report folder secured via Report Folder Sharing.
