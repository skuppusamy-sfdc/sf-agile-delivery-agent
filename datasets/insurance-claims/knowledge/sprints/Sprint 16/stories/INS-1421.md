## Description

As a Broker Relations Coordinator, I need a production performance report showing each licensed broker's total active policy count, total written premium, and new business submitted in the current quarter so that I can identify top-performing brokers for incentive programs and flag underperforming brokers for coaching outreach.

The report must be filterable by broker license state, LOB, and appointment status.

## Acceptance Criteria

Given a Broker Relations Coordinator accesses the 'Broker Production Summary' report,
When the report loads,
Then it displays one row per broker (Account record with RecordType = 'Broker') with the following columns:
  - Account.Name (Broker Name)
  - Account.License_State__c
  - Account.Appointment_Status__c
  - COUNT of Policy__c.Id WHERE Policy__c.Broker__c = Account.Id AND Policy__c.Status__c = 'Active'
  - SUM of Policy__c.Annual_Premium__c WHERE Policy__c.Status__c = 'Active'
  - COUNT of Policy__c.Id WHERE Policy__c.Effective_Date__c = THIS_QUARTER (New Business)

Given the report is displayed,
When the Broker Relations Coordinator applies the 'License State' filter,
Then only brokers with Account.License_State__c matching the selected state are shown.

Given the report is displayed,
When filtered by Account.Appointment_Status__c = 'Active',
Then brokers with 'Pending', 'Suspended', or 'Terminated' appointment status are excluded.

Given the report runs,
When a broker has zero active policies,
Then they are still included in the report with COUNT = 0 and SUM = $0.00 (no implicit filter excluding zero-production brokers).

Given the report is scheduled,
When the report runs automatically every Monday at 6:00 AM,
Then results are emailed to the Broker Relations Coordinator role queue.

## Solution

Create a custom Salesforce Report Type with Account (Broker) as the primary object and Policy__c as a related object (via Policy__c.Broker__c lookup). Use a summary report with grouping by Account.Name. New Business count implemented as a bucket field on Policy__c.Effective_Date__c filtering to current quarter. Filters for License State and Appointment Status implemented as standard report filters. Zero-production brokers included by using 'Show All Accounts with or without Policies' join type in the Report Type definition. Scheduled report delivery configured via Salesforce Report Subscription to the Broker Relations Coordinator role queue. Report added as a table component to the Broker Management Lightning dashboard.
