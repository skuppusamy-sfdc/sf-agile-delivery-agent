## Description

As a Provider Network Specialist, I need a Salesforce report and dashboard component that surfaces all Provider_Network_Agreement__c records expiring within the next 90 days, grouped by contract tier and network region, so that I can proactively initiate renewal workflows and prevent unintended provider terminations due to lapsed agreements.

This story extends INS-1438 (PNA lifecycle management flow) by providing the analytics visibility layer for the renewal queue.

## Acceptance Criteria

Given a Provider Network Specialist accesses the 'Provider Contracting' report folder,
When they open the 'PNA Renewal Pipeline — Next 90 Days' report,
Then the report returns all Provider_Network_Agreement__c records where:
  - PNA__c.Contract_End_Date__c >= TODAY AND PNA__c.Contract_End_Date__c <= TODAY+90
  - PNA__c.Status__c NOT IN ('Terminated', 'Superseded')
Grouped by PNA__c.Contract_Tier__c and PNA__c.Network_Region__c.

Given the report is generated,
When displayed,
Then each row must show: Provider__c (lookup name, NPI), PNA__c.Name, PNA__c.Contract_End_Date__c, PNA__c.Associated_Fee_Schedule__c (lookup name), PNA__c.Renewal_Status__c, PNA__c.Contract_Owner__c.

Given a PNA record has PNA__c.Renewal_Status__c = 'Renewal Initiated',
When it appears in the report,
Then the row is highlighted in green via conditional formatting.

Given a PNA record has PNA__c.Contract_End_Date__c <= TODAY+30 AND PNA__c.Renewal_Status__c = 'Pending',
When it appears in the report,
Then the row is highlighted in red.

Given the report is added to the 'Provider Network Ops' dashboard,
When the dashboard renders,
Then a bar chart displays count of expiring PNAs by Network_Region__c for the 90-day window.

## Solution

Create a Salesforce Reports & Dashboards solution: custom report type on Provider_Network_Agreement__c with related Provider__c (Person Account). Report uses relative date filter (Contract_End_Date__c NEXT_N_DAYS:90). Conditional highlighting rules configured on Renewal_Status__c and Contract_End_Date__c columns. Dashboard component uses a vertical bar chart widget. Report folder sharing set to 'Provider Network Ops' public group. No Apex required — pure declarative reporting. Schedule report to email Provider Network Specialist role daily at 7 AM.
