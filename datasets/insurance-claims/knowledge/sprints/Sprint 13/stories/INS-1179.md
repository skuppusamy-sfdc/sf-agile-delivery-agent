## Description

As a Provider Network Specialist, I need an OmniScript-driven PNA auto-renewal workflow that generates a pre-populated renewal agreement via Conga CLM and routes it for e-signature via DocuSign so that contract renewals are processed without manual document preparation, reducing turnaround time from 14 days to under 3 days.

## Acceptance Criteria

Given a Provider_Network_Agreement__c record where PNA.Termination_Date__c is within 90 days and PNA.Auto_Renewal_Eligible__c = true,
When the nightly scheduled flow executes,
Then PNA.Renewal_Status__c must be set to 'Renewal Initiated' and the PNA auto-renewal OmniScript must be launched in the background via Integration Procedure.

Given the OmniScript launches,
When the Integration Procedure calls the Conga CLM API,
Then a renewal contract document must be generated using the template mapped to PNA.Contract_Type__c and PNA.LOB__c, pre-populated with Provider__c.Name, Provider__c.NPI__c, Provider__c.TIN__c, PNA.Network_Name__c, PNA.Effective_Date__c (set to current PNA.Termination_Date__c + 1 day), and all Fee_Schedule__c.CPT_Code__c rates linked to the current PNA.

Given the Conga document is generated successfully,
When the DocuSign envelope is created,
Then the envelope must be sent to Provider__c.Contract_Signatory_Email__c with a 7-day expiration and DocuSign tag mapped to the signature block; PNA.DocuSign_Envelope_ID__c must be populated.

Given the provider completes e-signature in DocuSign,
When the DocuSign status callback webhook fires,
Then PNA.Renewal_Status__c must update to 'Executed', PNA.Termination_Date__c must be extended by PNA.Renewal_Term_Months__c, and a ContentVersion record must be created attaching the signed PDF to the PNA record.

Given the DocuSign envelope expires without signature after 7 days,
When the expiration webhook fires,
Then PNA.Renewal_Status__c must be set to 'Renewal Lapsed', a Task must be created assigned to the owning Provider Network Specialist, and a notification email must be sent referencing PNA.Contract_Number__c.

-Given PNA.Contract_Type__c = 'Legacy TPA Agreement', the system should bypass DocuSign and use wet signature process- (deprecated; all contract types now require DocuSign per compliance mandate effective Sprint 12).

## Solution

Build an OmniStudio Integration Procedure (PNA_AutoRenewal_IP) orchestrating: (1) DataRaptor Extract to gather PNA and related Provider__c, Fee_Schedule__c data; (2) HTTP Action to Conga CLM REST API for document generation; (3) HTTP Action to DocuSign eSignature REST API to create and send envelope, storing envelope ID back to PNA via DataRaptor Load. A Scheduled Flow runs nightly to query qualifying PNA records and invoke the Integration Procedure via Apex (OmnistudioIntegrationProcedureService). DocuSign status callbacks handled by a custom REST endpoint (DocuSignCallbackController.cls) that publishes a Platform Event (PNA_Signature_Event__e) consumed by a Record-Triggered Flow to update PNA fields and create ContentVersion.
