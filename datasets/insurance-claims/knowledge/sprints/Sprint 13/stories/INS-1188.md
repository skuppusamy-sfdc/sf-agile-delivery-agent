## Description

As a Claims Operations Manager, I need Agentforce to analyze incoming Claim__c records against fraud risk indicators and surface a next-best-action (NBA) recommendation to refer the claim to the SIU (Special Investigations Unit) so that high-risk claims are flagged proactively before payment is disbursed, reducing fraudulent payment exposure.

## Acceptance Criteria

Given a Claim__c record where Claim__c.Status__c transitions to 'Pending Adjudication',
When the Agentforce Claims AI agent evaluates the record,
Then it must score the claim using the SIU_Risk_Score__c field (0–100) based on configured risk indicators including billed amount variance, provider billing pattern anomalies, and duplicate service date detection.

Given Claim__c.SIU_Risk_Score__c >= 75,
When the NBA engine evaluates the score,
Then a Next Best Action recommendation card labeled 'Refer to SIU' must appear on the Claim__c record page with Agentforce rationale text summarizing the top 3 contributing risk factors.

Given the Claims Operations Manager accepts the SIU referral NBA,
When the action is confirmed,
Then Claim__c.Status__c must be set to 'SIU Review', Claim__c.SIU_Referred_Date__c = TODAY(), and a Task must be created assigned to the SIU queue with Subject = 'SIU Review Required — ' + Claim__c.Name.

Given the Claims Operations Manager dismisses the SIU referral NBA,
When the dismissal is logged,
Then NBA_Dismissal__c record must be created with NBA_Dismissal__c.Reason__c captured from a required picklist and Claim__c.SIU_Override_Reason__c populated for audit.

Given Claim__c.SIU_Risk_Score__c < 75,
When the NBA engine evaluates the score,
Then no SIU referral card must be displayed and standard adjudication flow must proceed uninterrupted.

## Solution

Configure Agentforce Agent Action using Einstein Next Best Action with Strategy built in Strategy Builder. Risk scoring logic implemented in Apex invocable class SIURiskScorer called as a Flow element within the NBA Strategy. Risk indicators sourced from Claim__c, Claim_Line__c, and historical Payment__c aggregates via SOQL. NBA recommendation card rendered as a custom LWC 'siuReferralCard' embedded in Claim__c Lightning record page via Dynamic Actions. Task creation on acceptance handled by Record-Triggered Flow 'SIU_Referral_Task_Flow'. NBA_Dismissal__c object created with Master-Detail to Claim__c. Permission Set 'Agentforce_Claims_AI_User' grants access to NBA features. Agentforce topic configuration scoped to Claims Operations Manager profile.
