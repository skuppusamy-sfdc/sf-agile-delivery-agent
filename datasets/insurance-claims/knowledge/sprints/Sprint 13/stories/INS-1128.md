## Description

As a Provider Relations Specialist, I need a unified Provider 360 FlexCard on the Provider__c (Person Account) record that surfaces real-time credentialing status, active PNA agreements, network participation summary, and open PCNs so that I can resolve provider inquiries without navigating across multiple objects.

## Acceptance Criteria

Given a Provider__c record is opened by a user with the 'Provider_Relations_PS' Permission Set Group,
When the Provider 360 FlexCard loads,
Then the card must display Provider__c.NPI__c, Provider__c.TIN__c, Provider__c.Credentialing_Status__c, and Provider__c.Network_Participation__c in the header band.

Given an active Credentialing_Case__c exists linked to the provider,
When the FlexCard Credentialing panel renders,
Then Credentialing_Case__c.Status__c, Credentialing_Case__c.CAQH_Profile_ID__c, and Credentialing_Case__c.Expiration_Date__c must be shown with a red badge when Expiration_Date__c is within 30 days of today.

Given one or more Provider_Network_Agreement__c records exist with PNA.Status__c = 'Active',
When the Network Agreements tab is selected on the FlexCard,
Then all active PNAs must be listed showing PNA.Effective_Date__c, PNA.Termination_Date__c, PNA.LOB__c, and PNA.Fee_Schedule__r.Name.

Given an open Provider_Change_Notification__c (PCN) record exists with PCN.Status__c != 'Closed',
When the Alerts section renders,
Then each open PCN must display PCN.Change_Type__c, PCN.Submitted_Date__c, and a hyperlink to the PCN record.

-Given the provider has a linked Facility__c, display the facility TIN on the header band-
(Superseded: Facility TIN now displayed on separate Facility 360 card per INS-1101)

Given the FlexCard is viewed on a mobile form factor,
When the viewport is less than 768px,
Then the card must collapse PNA and PCN panels into accordion sections without horizontal scroll.

## Solution

Implement using OmniStudio FlexCard with child FlexCards for Credentialing, PNA, and PCN panels. Use DataRaptor Extract actions to query Credentialing_Case__c, Provider_Network_Agreement__c, and Provider_Change_Notification__c via SOQL with provider Account ID as the key. Configure conditional styling on Expiration_Date__c using a FlexCard State with a formula evaluating TODAY() - 30. Deploy via OmniStudio Deployment Package. Assign visibility via Permission Set Group 'Provider_Relations_PS'. Mobile responsiveness handled through FlexCard responsive layout settings.
