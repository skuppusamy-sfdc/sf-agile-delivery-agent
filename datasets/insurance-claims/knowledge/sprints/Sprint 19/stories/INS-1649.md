## Description

As a Broker Relations Coordinator, I need the system to automatically activate a Group_Service_Agreement__c (GSA) record and trigger a member eligibility file request to the employer's TPA when a DocuSign envelope for the GSA is marked as 'Completed' so that group onboarding timelines are met and manual activation steps are eliminated, reducing average onboarding cycle time from 5 days to same-day.

## Acceptance Criteria

Given a Group_Service_Agreement__c record with GSA__c.DocuSign_Envelope_Status__c changing to 'Completed' via the DocuSign Status Update webhook,
When the Record-Triggered Flow (RTF_GSAActivationOnExecution) fires,
Then GSA__c.Status__c is updated to 'Active', GSA__c.Execution_Date__c = TODAY(), GSA__c.Activated_By__c = the running user's Id, and GSA__c.Effective_Date__c is set to the first day of the following month if Execution_Date__c is after the 15th, or the first day of the current month if on or before the 15th.

Given GSA__c.Status__c transitions to 'Active',
When the activation flow completes,
Then an outbound Platform Event (GSAActivated__e) is published with payload: GSA_Id__c, Employer_Account_Id__c, TPA_Contact_Id__c, Effective_Date__c, Group_Number__c.

Given the GSAActivated__e Platform Event is received by the subscribing integration flow,
When the eligibility file request is triggered,
Then an Eligibility_File_Request__c record is created with: EFR__c.GSA__c = GSA__c.Id, EFR__c.Request_Type__c = 'Initial Enrollment', EFR__c.Requested_Date__c = TODAY(), EFR__c.Status__c = 'Sent to TPA', EFR__c.Expected_Receipt_Date__c = TODAY() + 5 (business days).

Given the GSA activation completes,
When all records are created,
Then the assigned Broker Relations Coordinator receives an automated email notification (template 'GSA_Activation_Confirmation') and a Chatter post is made on the GSA record: 'GSA [GSA__c.Group_Number__c] activated effective [Effective_Date__c]. Eligibility file request sent to TPA.'

Given a GSA where GSA__c.TPA_Contact__c is null at time of activation,
When the flow attempts to publish the Platform Event,
Then the flow logs an error to Error_Log__c, sets GSA__c.Activation_Error__c = 'TPA Contact Missing — Manual Follow-up Required', and creates a high-priority Task assigned to the Broker Relations Coordinator.

## Solution

Record-Triggered Flow 'RTF_GSAActivationOnExecution' on Group_Service_Agreement__c, after-save, entry condition: DocuSign_Envelope_Status__c changed to 'Completed'. Flow uses Decision element for effective date calculation logic. Platform Event GSAActivated__e defined with required payload fields. Subscriber flow handles Eligibility_File_Request__c creation and email notification via OrgWide Email Address. DocuSign webhook configured via DocuSign for Salesforce managed package to write envelope status back to GSA__c.DocuSign_Envelope_Status__c. Error handling via Fault Path in Flow writing to Error_Log__c custom object. Permission Set 'BrokerRelations_PSG' updated to include read access to Eligibility_File_Request__c.
