## Description

As a Compliance Officer, I need a Salesforce Lightning Dashboard dedicated to Network_Termination_Case__c records so that I can monitor active terminations, track regulatory notice compliance (e.g., 60-day advance notice requirements), identify overdue cases, and report on termination reasons by LOB — all from a single pane of glass without requiring manual data exports.

## Acceptance Criteria

Given the 'Network Termination Compliance Dashboard' is accessed by a user with the 'Compliance_Officer' Permission Set Group,
When the dashboard loads,
Then it displays the following components: (1) Donut chart — Network_Termination_Case__c records by Network_Termination_Case__c.Termination_Reason__c, (2) Bar chart — open termination cases by Network_Termination_Case__c.LOB__c, (3) Metric tile — count of cases where Network_Termination_Case__c.Notice_Sent_Date__c is null AND Network_Termination_Case__c.Status__c = 'Active' (Notice Not Sent), (4) Table — cases where Network_Termination_Case__c.Effective_Termination_Date__c <= TODAY() + 30 days (upcoming terminations), (5) Metric tile — average days between Network_Termination_Case__c.Case_Open_Date__c and Network_Termination_Case__c.Effective_Termination_Date__c.

Given the dashboard is displayed,
When a user clicks a bar chart segment,
Then the underlying report filters to the selected LOB and opens in a new tab.

Given a user without the 'Compliance_Officer' Permission Set Group,
When they attempt to access the dashboard via direct URL,
Then they are redirected and see a 'Insufficient Privileges' message — no Network_Termination_Case__c data is exposed.

-Given the dashboard includes a map component showing provider termination density by state (deprecated — licensing cost for map component not approved in Phase 2 budget)-

Given the dashboard is scheduled,
When the weekly schedule runs (every Monday at 7:00 AM local),
Then a dashboard snapshot is emailed to the 'Compliance_Officers' public group.

## Solution

Build 5 CRM Analytics (Salesforce Reports & Dashboards) report components sourced from Network_Termination_Case__c object with appropriate filters. Dashboard folder shared with 'Compliance_Officer' Permission Set Group via folder sharing settings. Row-level security enforced via Sharing Rules on Network_Termination_Case__c scoped to Compliance Officer role hierarchy. Dashboard subscription configured via standard Salesforce dashboard scheduling to the 'Compliance_Officers' public group. Click-through drill-down enabled by linking each dashboard component to its underlying source report. All reports stored in 'Compliance — Network Termination' report folder with restricted access.
