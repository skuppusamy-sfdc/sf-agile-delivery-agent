## Description

As a Compliance Officer, I need a Salesforce-native regulatory reporting dashboard that aggregates claim metrics by LOB, state, and adjudication outcome so that monthly Department of Insurance (DOI) filings can be prepared and submitted without manual data extraction from multiple systems. Extends INS-1554.

## Acceptance Criteria

Given the Compliance Officer navigates to the 'DOI Monthly Reporting' app page,
When the page loads,
Then a CRM Analytics dashboard must display: total claims by Claim__c.State_of_Service__c, average days-to-adjudication (Claim__c.Adjudication_Date__c minus Claim__c.FNOL_Date__c), denial rate by Denial_Reason__c picklist value, and total paid amount from Payment__c.Amount__c grouped by LOB.

Given the reporting period selector is set to a specific month/year,
When the Compliance Officer clicks 'Generate Report',
Then all dashboard widgets must filter to that period using Claim__c.Service_Date__c and the export must produce a flat CSV matching the state NAIC reporting template column order.

Given a state requires prompt-pay compliance reporting,
When claims with Claim__c.Received_Date__c are evaluated,
Then any claim where Payment__c.Payment_Date__c exceeds Claim__c.Received_Date__c + 30 calendar days must be flagged with Prompt_Pay_Violation__c = true and counted in a dedicated dashboard tile.

Given the dashboard is accessed,
When a user without the 'Compliance Officer' Permission Set Group attempts to view it,
Then the page must display an 'Access Denied' message and log the access attempt to Compliance_Access_Log__c.

Given the monthly report is finalized,
When the Compliance Officer clicks 'Submit to DOI',
Then a DocuSign envelope must be triggered via API containing the generated PDF report, pre-populated with the authorized signatory from Org_Compliance_Settings__mdt.

## Solution

1. Create CRM Analytics (Tableau CRM) dataset recipe joining Claim__c, Payment__c, Claim_Line__c, and EOB__c objects with scheduled daily refresh. 2. Build CRMA dashboard with SAQL-based widgets for each KPI tile; parameterize reporting period via dashboard filter. 3. Apex-scheduled job sets Claim__c.Prompt_Pay_Violation__c = true nightly via batch query. 4. 'Generate Report' button triggers Apex Controller invoking Conga Composer template to produce NAIC-formatted PDF. 5. DocuSign REST API callout from Apex sends envelope to signatory email from Custom Metadata Type (Org_Compliance_Settings__mdt). 6. Permission Set Group 'Compliance_Reporting_PSG' gates dashboard access; before-render LWC checks PSG membership and logs denied attempts.
