## Description

As a Provider Relations Specialist, I need automated compliance alerts and audit log entries when a Provider_Network_Agreement__c (PNA) is within 90, 60, or 30 days of its Contract_End_Date__c so that we can proactively renegotiate contracts and maintain regulatory network adequacy standards before agreements lapse.

## Acceptance Criteria

Given a PNA record has Contract_End_Date__c set and PNA_Status__c = 'Active', When a nightly scheduled job runs, Then the system must identify all PNAs where Contract_End_Date__c is within 90 days of today and create a Provider_Contract_Alert__c record with Alert_Threshold__c = '90 Days', linked PNA__c, Provider__c (Person Account), and Assigned_To__c = the Provider Relations Specialist on the PNA.

Given a Provider_Contract_Alert__c is created at the 60-day threshold, When the alert record is inserted, Then a Task must be auto-created with Subject = 'PNA Renewal Required - 60 Day Notice', Due_Date__c = today + 10 days, assigned to the same Provider Relations Specialist, and linked to the PNA via WhatId.

Given a PNA reaches its Contract_End_Date__c without PNA_Status__c transitioning to 'Renewed' or 'Extended', When the nightly job detects the lapse, Then PNA_Status__c must be set to 'Expired', a PNA_Audit_Log__c record must be created with Event_Type__c = 'Contract Lapsed', and all active Claim__c records linked to that provider with Claim__c.Status__c = 'In Review' must have a warning flag Compliance_Hold__c = true applied.

Given the Provider Relations Specialist views a Provider 360 FlexCard, When the associated PNA has an active alert, Then a banner must display the alert threshold, days remaining, and a 'Initiate Renewal' quick action button that launches the PNA renewal OmniScript.

-Given alerts are sent via legacy email template 'PNA_Expiry_Notice_v1'- (deprecated: replaced by Salesforce Notification Builder per compliance team request)

Given the audit log is queried, When a Compliance Officer filters PNA_Audit_Log__c by Event_Type__c = 'Contract Lapsed' and Date_Range__c, Then all matching records must be returned and exportable.

## Solution

Scheduled Apex class 'PNAExpiryComplianceJob' runs nightly at 2:00 AM EST, querying Provider_Network_Agreement__c with SOQL date literals. Apex creates Provider_Contract_Alert__c and Task records in bulk using Database.insert with allOrNone=false for resilience. A Record-Triggered Flow on Provider_Network_Agreement__c (after-update, Status__c changed to 'Expired') fires a subflow to bulk-update linked Claim__c.Compliance_Hold__c via a related list update. PNA_Audit_Log__c uses the same immutability pattern as INS-1555 (Apex Trigger blocks DML). FlexCard 'Provider360Card' updated with conditional rendering expression for the alert banner. Salesforce Notification Builder custom notification type 'PNA_Expiry_Alert' replaces legacy email templates.
