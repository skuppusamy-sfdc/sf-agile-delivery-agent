## Description

As a Provider Network Specialist, I need the system to automatically generate a Provider Change Notification (PCN) document and enforce a 90-day advance notice compliance window when a Network_Termination_Case__c is initiated for a contracted provider, so that Acme Insurance satisfies state-mandated advance termination notice requirements and protects members from unexpected network disruptions.

## Acceptance Criteria

Given a Network_Termination_Case__c is created with Termination_Type__c = 'Without Cause', When the record is saved, Then the system must calculate Earliest_Effective_Termination_Date__c = CreatedDate + 90 days and prevent any Termination_Effective_Date__c value earlier than this calculated date, displaying validation error: 'Without-cause terminations require a minimum 90-day advance notice per state regulatory requirements.'

Given the Network_Termination_Case__c passes the 90-day validation, When Case_Status__c is set to 'Notice Issued', Then a Provider_Change_Notification__c (PCN) record must be auto-created with: PCN_Type__c = 'Network Termination', Linked_Provider__c, Linked_PNA__c, Notice_Issue_Date__c = today, Effective_Date__c = Termination_Effective_Date__c, and PCN_Status__c = 'Pending Delivery'.

Given the PCN record is created, When the record is inserted, Then a DocuSign envelope must be sent via the DocuSign API to the provider's Provider__c.Email__c address containing the termination notice template 'PCN_Termination_Notice_v3', with envelope tracking stored on PCN__c.DocuSign_Envelope_Id__c.

Given Termination_Type__c = 'For Cause' (e.g., fraud, license revocation), When the Network_Termination_Case__c is saved with this type, Then the 90-day notice requirement must be waived, Earliest_Effective_Termination_Date__c = CreatedDate, and PCN_Status__c must be set to 'Expedited — For Cause', triggering an immediate SIU notification Task.

Given a PCN is in 'Pending Delivery' status, When 5 business days pass without DocuSign envelope completion, Then an escalation Task must be created for the Provider Network Specialist and PCN_Status__c updated to 'Delivery Escalated'.

Given the termination becomes effective (today = Termination_Effective_Date__c), When the nightly compliance job runs, Then Provider__c.Network_Status__c must be set to 'Terminated', the linked PNA's PNA_Status__c must be set to 'Terminated', and all active Claim__c records in 'Pending' status for the provider must have Compliance_Hold__c = true with Compliance_Hold_Reason__c = 'Provider Network Terminated'.

## Solution

Validation Rule on Network_Termination_Case__c enforces 90-day minimum for 'Without Cause' terminations. Record-Triggered Flow (after-insert, after-update on Case_Status__c = 'Notice Issued') creates PCN record and invokes Apex class 'DocuSignPCNEnvelopeService' for envelope dispatch via DocuSign REST API Named Credential. Scheduled Apex 'PCNDeliveryEscalationJob' runs nightly checking for undelivered envelopes older than 5 business days using Business_Hours.diff(). Separate Scheduled Apex 'NetworkTerminationEffectiveDateJob' handles termination date enforcement, bulk-updating Provider__c, PNA__c, and Claim__c records. For-Cause path uses a decision element in the Record-Triggered Flow to bypass date validation and route to SIU Task creation. Custom Metadata 'Termination_Config__mdt' stores notice period by Termination_Type__c for future configurability.
