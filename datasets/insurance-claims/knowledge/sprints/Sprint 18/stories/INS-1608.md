## Description

As a Compliance Officer, I need a scheduled quarterly data extract conforming to the NAIC Quarterly Statement format that aggregates Claim__c, Payment__c, and Policy__c data by LOB so that the organization can submit accurate regulatory reports to state Departments of Insurance without manual data wrangling in spreadsheets.

## Acceptance Criteria

Given the reporting period end date is the last calendar day of a quarter,
When the scheduled Apex job 'NAIC_Quarterly_Extract_Batch' executes at 02:00 UTC on the first day of the following month,
Then a Regulatory_Report__c record must be created with: Report_Period__c, LOB__c, Total_Claims_Filed__c, Total_Claims_Paid__c, Total_Incurred_Loss__c, Total_Reserves__c, Report_Status__c = 'Generated', and a CSV ContentVersion attachment in NAIC flat-file format.

Given the extract is generated,
When Total_Claims_Filed__c is calculated,
Then it must count all Claim__c records where FNOL_Date__c falls within the report quarter AND Claim__c.LOB__c matches the report's LOB__c, grouped by state (Policy__c.State__c).

Given a Payment__c record has Payment_Type__c = 'Void' or 'Reversal',
When the extract aggregates Total_Claims_Paid__c,
Then voided and reversed payments must be excluded from the paid totals and flagged in a separate Regulatory_Report_Exception__c child record.

Given the extract job completes,
When Report_Status__c = 'Generated',
Then an automated email notification must be sent to all users with the 'DOI_Reporting' Permission Set listing the report period, LOB, record counts, and a link to the Regulatory_Report__c record.

Given a Compliance Officer reviews the report,
When they click 'Approve for Submission',
Then Report_Status__c must transition to 'Approved', Approved_By__c and Approved_Date__c must be stamped, and a DocuSign envelope must be triggered for the Chief Compliance Officer's e-signature via the DocuSign Apex Toolkit.

-Given the extract includes provider TIN data in column 47 of the flat file- (deprecated: TIN masked per PII policy INS-1610 requirement)

Given Report_Status__c = 'Approved',
When the submission deadline is within 5 calendar days,
Then a Task must be auto-created and assigned to the Compliance Officer with Subject = 'DOI Submission Due: [LOB] [Quarter]' and ActivityDate set to the deadline.

## Solution

Develop a Schedulable Apex class 'NAIC_Quarterly_Extract_Batch' implementing Database.Batchable and Database.Stateful to aggregate data across Claim__c, Payment__c, Policy__c, and Claim_Line__c objects in chunked SOQL queries of 200 records. A custom metadata type 'NAIC_Report_Config__mdt' will store LOB-to-NAIC-schedule mappings and column definitions, enabling configuration without code deployment. The CSV ContentVersion is assembled using a StringBuilder pattern in the finish() method. A Record-Triggered Flow on Regulatory_Report__c handles the 'Approve for Submission' status transition, DocuSign invocation via named Flow action, and Task creation. A Permission Set Group 'DOI_Reporting_Access' controls report visibility. Scheduling is managed via a custom Setup page backed by a System.schedule() invocation.
