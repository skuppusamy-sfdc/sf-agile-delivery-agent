## Description

As a Provider Network Specialist, I need automated notifications triggered at 90, 60, and 30 days before a Provider_Network_Agreement__c (PNA) expires so that I can proactively initiate contract renewals and avoid unintended network gaps that could impact claims adjudication.

Notifications must be sent to the assigned Provider Relations Specialist and the Provider Relations Manager queue.

## Acceptance Criteria

Given a Provider_Network_Agreement__c record where PNA__c.Status__c = 'Active' and PNA__c.Contract_End_Date__c is within 90 calendar days of today,
When the nightly scheduled Apex job runs,
Then a Platform Event 'PNA_Expiry_Alert__e' is published with fields: PNA_Id__c, Provider_Id__c, Days_Until_Expiry__c, and Alert_Threshold__c = '90'.

Given the PNA_Expiry_Alert__e Platform Event is published,
When the trigger-based Flow subscribes to the event,
Then a Task is created on the Provider_Network_Agreement__c record with Subject = 'PNA Renewal Required - [X] Days Remaining', Status = 'Not Started', Priority = 'High', and OwnerId set to PNA__c.Assigned_Specialist__c.

Given the Task is created for a 30-day threshold,
When the Task is created,
Then an Email Alert is sent to PNA__c.Assigned_Specialist__c and the 'Provider_Relations_Manager' queue using the 'PNA_30Day_Expiry_Alert' email template.

Given a PNA__c where PNA__c.Renewal_In_Progress__c = true,
When the scheduled job evaluates that record,
Then no Platform Event is published and no Task or email is generated for that record.

Given a PNA__c where PNA__c.Status__c != 'Active',
When the scheduled job runs,
Then the record is excluded from all expiry notification processing.

## Solution

Implement a Schedulable Apex class 'PNAExpiryNotificationScheduler' running nightly at 6 AM UTC. The class queries Provider_Network_Agreement__c with filters on Status__c = 'Active', Renewal_In_Progress__c = false, and Contract_End_Date__c within 90 days. For each qualifying record, publish PNA_Expiry_Alert__e Platform Events in bulk using EventBus.publish(). Build a Record-Triggered Flow subscribed to PNA_Expiry_Alert__e that creates Tasks and triggers the Email Alert action. Email template 'PNA_30Day_Expiry_Alert' built in Salesforce with merge fields. Schedule the Apex job via Setup > Apex Classes > Schedule.
