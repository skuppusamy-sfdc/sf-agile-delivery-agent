## Description

As a Provider Network Specialist, I need the ability to associate a Group_Service_Agreement__c (GSA) with one or more Provider_Network_Agreement__c (PNA) records and have the linked GSA summary surfaced on the Provider 360 FlexCard (INS-0757) so that I can confirm which employer groups a provider is contracted to serve and validate GSA-specific fee schedule applicability during claims adjudication.

## Acceptance Criteria

Given a Group_Service_Agreement__c record exists with GSA.Status__c = 'Active',
When a Provider Network Specialist navigates to a PNA record and selects 'Link GSA',
Then a lookup field PNA.Group_Service_Agreement__c must allow selection of active GSA records filtered by GSA.Effective_Date__c <= TODAY and GSA.Expiration_Date__c >= TODAY.

Given a GSA is linked to a PNA (PNA.Group_Service_Agreement__c is populated),
When the Provider 360 FlexCard (INS-0757) loads for the associated provider,
Then a 'Group Agreements' section must display: GSA.Name, GSA.Employer_Group__c, GSA.Effective_Date__c, GSA.Expiration_Date__c, GSA.Contracted_Network_Tier__c.

Given a GSA.Expiration_Date__c < TODAY,
When the GSA is linked to an active PNA,
Then a validation rule must prevent saving PNA.Group_Service_Agreement__c with an expired GSA, returning error: 'Cannot link an expired Group Service Agreement. Please select an active GSA or renew the existing agreement.'

Given a new GSA is linked to a PNA,
When the save action completes,
Then a Platform Event 'GSA_PNA_Linked__e' must be published containing GSA.Id, PNA.Id, and Provider__c.NPI__c for downstream notification to the Broker Relations Coordinator.

Given the Platform Event 'GSA_PNA_Linked__e' is received,
When the event trigger fires,
Then a Task must be created assigned to the Broker Relations Coordinator associated with GSA.Broker__c with Subject = 'Review GSA-PNA Linkage: [GSA.Name]' and Due Date = TODAY + 5.

## Solution

Add GSA lookup field (Group_Service_Agreement__c) to Provider_Network_Agreement__c object. Build Validation Rule on PNA to block expired GSA linkage. Create Record-Triggered Flow on PNA (after insert/update) to publish Platform Event 'GSA_PNA_Linked__e' when Group_Service_Agreement__c is set. Build Platform Event-Triggered Flow to create Task for Broker Relations Coordinator. Update DR_GetActivePNAs DataRaptor in INS-0757 FlexCard to include GSA child fields. Update OmniStudio FlexCard 'Provider360Card' with new 'Group Agreements' state block. Add field-level security for new GSA lookup to 'Provider_Network_PSG' Permission Set.
