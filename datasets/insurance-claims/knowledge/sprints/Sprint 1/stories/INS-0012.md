## Description

**Bug Report — Extends INS-0007 (Policy__c schema deployment)**

**Steps to Reproduce:**
1. Navigate to any existing Policy__c record where Status__c = 'Active' and Termination_Date__c is populated.
2. Click the standard Salesforce 'Clone' button from the record detail page.
3. In the cloned record form, clear the Termination_Date__c field (intentionally creating an open-ended policy for a renewal scenario).
4. Set Effective_Date__c to today's date.
5. Click Save.

**Expected Behavior:**
The cloned Policy__c record saves successfully when Termination_Date__c is NULL, as an open-ended active policy is a valid business scenario for month-to-month or evergreen policy types.

**Actual Behavior:**
Validation rule 'VR_Policy_Term_Date_After_Eff_Date' fires with error: 'Termination Date cannot precede Effective Date' even though Termination_Date__c is NULL (blank), not a date that precedes Effective_Date__c. The validation rule formula does not account for the NULL case.

**Environment:** Acme Insurance - Phase 1 Sandbox (API v59.0)
**Severity:** High — blocks Policy renewal cloning workflow used by Underwriting Analysts daily.
**Reported By:** Emily Watson (UAT session 2024-01-14)

## Acceptance Criteria

as per Description — validation rule formula must be corrected to only fire when BOTH Effective_Date__c AND Termination_Date__c are non-null AND Termination_Date__c < Effective_Date__c.

## Solution

Update validation rule 'VR_Policy_Term_Date_After_Eff_Date' on Policy__c. Current (broken) formula: Termination_Date__c < Effective_Date__c. Corrected formula: AND(NOT(ISBLANK(Termination_Date__c)), NOT(ISBLANK(Effective_Date__c)), Termination_Date__c < Effective_Date__c). Deploy via changeset to sandbox first, execute regression test covering: (1) NULL Termination_Date__c — should save, (2) Termination_Date__c > Effective_Date__c — should save, (3) Termination_Date__c < Effective_Date__c — should block. Update corresponding Apex test class 'PolicyValidationTest' to add test method 'testCloneWithNullTerminationDate'. Promote to UAT sandbox after test pass.
