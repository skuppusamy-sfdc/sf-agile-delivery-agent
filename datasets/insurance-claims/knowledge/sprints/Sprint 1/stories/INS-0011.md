## Description

As a Provider Network Specialist, I need distinct Record Types on Provider_Network_Agreement__c (PNA) for Individual Provider contracts versus Facility contracts, each with tailored page layouts and picklist values, so that the contracting workflow captures the correct data elements for each agreement type and reduces data entry errors during PNA creation.

## Acceptance Criteria

Given the metadata deployment is complete,
When a Provider Network Specialist creates a new PNA record,
Then they are prompted to select one of two Record Types: 'Individual_Provider_PNA' or 'Facility_PNA'.

Given Record Type 'Individual_Provider_PNA' is selected,
When the new PNA page layout renders,
Then the layout includes: NPI__c (required), Primary_Specialty__c (required), Secondary_Specialty__c, Board_Certification__c (Checkbox), Solo_Practice__c (Checkbox), and the Facility__c lookup field is hidden.

Given Record Type 'Facility_PNA' is selected,
When the new PNA page layout renders,
Then the layout includes: Facility_NPI__c (required), TIN__c (required), Bed_Count__c (Number), Accreditation_Body__c (Picklist: JCAHO|DNV|CIHQ|None), and the individual NPI__c and Board_Certification__c fields are hidden.

Given either Record Type,
When the PNA is saved,
Then Contract_Effective_Date__c is required, Status__c defaults to 'Pending Execution', and a DocuSign envelope is NOT auto-sent (DocuSign trigger is deferred to Phase 2 per INS-0022).

Given both Record Types exist,
When picklist values for Contract_Type__c are reviewed,
Then 'Individual_Provider_PNA' record type has values [Full Risk|Partial Risk|FFS|Capitation] and 'Facility_PNA' has values [Per Diem|Case Rate|DRG|Bundled Payment|FFS].

## Solution

Create two Record Types on Provider_Network_Agreement__c via Object Manager. Build separate page layouts 'PNA_Individual_Layout' and 'PNA_Facility_Layout' with appropriate field assignments. Configure Record Type picklist value sets for Contract_Type__c using restricted picklist to prevent cross-record-type values. Assign layouts to Record Types in the Page Layout Assignment matrix. Update Permission Set Groups (from INS-0008) to grant Record Type visibility for both types to PSG 'Provider_Network_Specialist' and 'Claims_Ops_Manager'. Validation rule added: IF RecordType.DeveloperName = 'Individual_Provider_PNA' THEN NPI__c != NULL enforced at save.
