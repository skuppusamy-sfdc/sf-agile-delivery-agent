## Description

As a Claims Operations Manager, I need an Agentforce Copilot action embedded on the Claim__c record page that generates a natural-language summary of the claim — including claimant details, FNOL date, current adjudication status, total billed vs. allowed amounts, active SIU flags, and COB coordination status — so that adjusters can rapidly orient themselves on complex claims during the hypercare period without reading through all related records manually.

## Acceptance Criteria

Given an adjuster opens any Claim__c record in the 'Under Review' or 'Adjudicated' status,
When they invoke the Agentforce Copilot panel and select 'Summarize Claim',
Then the Copilot action returns a structured natural-language summary within 8 seconds containing: claimant name (from Policy__c.Policyholder__c), Claim__c.FNOL_Received_Date__c, Claim__c.Status__c, Claim__c.Total_Billed_Amount__c, Claim__c.Total_Allowed_Amount__c, and Claim__c.COB_Status__c.

Given Claim__c.SIU_Flag__c = true,
When the summary is generated,
Then the Copilot response includes a bold warning: '⚠ SIU REFERRAL ACTIVE — Exercise caution. Do not share claim details externally without SIU clearance.' and the summary is logged to a new Agentforce_Interaction_Log__c record with Interaction_Type__c = 'Claim Summary'.

Given the Copilot action is invoked,
When the underlying Apex Copilot Action class (ClaimSummaryAgentAction) executes,
Then it retrieves data only from Claim__c, Policy__c, Claim_Line__c (aggregate totals), and Explanation_of_Benefits__c — no external callouts are made during summarization.

Given a user without the 'Claims_AI_User' Permission Set assigned,
When they attempt to invoke 'Summarize Claim',
Then the Copilot panel displays: 'This action requires Claims AI access. Contact your administrator.' and no Agentforce_Interaction_Log__c is created.

Given the Agentforce summarization is complete,
When the adjuster clicks 'Copy to Clipboard',
Then the plain-text summary is copied and a Toast notification confirms: 'Claim summary copied — suitable for internal use only.'

## Solution

Configure an Agentforce Copilot Action ('Summarize Claim') in Einstein Copilot Studio, backed by an Apex class implementing the @InvocableMethod interface (ClaimSummaryAgentAction.cls). The Apex method accepts Claim__c ID as input, executes a SOQL query joining Claim__c, Policy__c, and aggregate Claim_Line__c data, and returns a structured ClaimSummaryResult wrapper object. The prompt template ('Claim_Summary_Prompt_Template') is configured in Einstein Generative AI Prompt Builder with merge fields mapped to the Apex output. SIU flag check performed in Apex pre-prompt with conditional injection of the warning string. Agentforce_Interaction_Log__c created via a Record-Triggered Flow on Einstein Copilot Session object post-action execution. Permission enforcement via Copilot Action visibility settings scoped to 'Claims_AI_User' Permission Set. 'Copy to Clipboard' implemented as an LWC utility button in the Copilot response card.
